## 核心讲义

积分有两副面孔：一副是「求导的逆运算」（不定积分），另一副是「无穷多个小量的累加极限」（定积分）。微积分基本定理揭示这两副面孔其实是同一个东西，这是整个微积分最深刻的结论之一，也是把「累加问题」化归为「求原函数」的关键。

### 不定积分与基本积分表

**定义.** 若 $F'(x) = f(x)$，则称 $F$ 是 $f$ 的一个**原函数**。$f$ 的全体原函数 $F(x)+C$ 称为 $f$ 的**不定积分**，记作
$$\int f(x)\,dx = F(x) + C,$$
其中 $C$ 是任意常数（因为常数求导为零，原函数不唯一，相差一个常数）。由拉格朗日中值定理可证：同一函数的任意两个原函数之差必为常数，所以「$+C$」概括了**全部**原函数。

不定积分与求导互为逆运算，所以每条求导公式倒过来就是一条积分公式。**基本积分表**（务必背熟）：
$$\int x^a\,dx = \frac{x^{a+1}}{a+1} + C\ (a\ne -1), \qquad \int \frac{1}{x}\,dx = \ln|x| + C,$$
$$\int e^x\,dx = e^x + C, \qquad \int \sin x\,dx = -\cos x + C, \qquad \int \cos x\,dx = \sin x + C,$$
$$\int \frac{1}{1+x^2}\,dx = \arctan x + C, \qquad \int \frac{1}{\sqrt{1-x^2}}\,dx = \arcsin x + C.$$

线性性质：$\displaystyle\int [a f(x) + b g(x)]\,dx = a\int f\,dx + b\int g\,dx$。注意 $\int\frac1x\,dx$ 的绝对值——它保证公式在 $x<0$ 时同样成立。

### 换元积分法

当被积函数不在基本表中，最常用的化简手段是**换元**。

**第一类换元（凑微分）.** 核心是把被积式凑成 $f(g(x))g'(x)\,dx = f(u)\,du$ 的形式：
$$\int f(g(x))\,g'(x)\,dx = \int f(u)\,du \quad (u = g(x)).$$
例如 $\int 2x\,e^{x^2}\,dx$，注意到 $d(x^2)=2x\,dx$，令 $u=x^2$ 即化为 $\int e^u\,du = e^{x^2}+C$。它本质是链式法则的逆用，关键在于「看出」哪一部分是某个内函数的微分。

**第二类换元（变量代换）.** 主动设 $x=\varphi(t)$，把 $x$ 换成新变量 $t$，常用于根式。例如遇到 $\sqrt{a^2 - x^2}$ 可设 $x = a\sin t$，利用 $1-\sin^2 t=\cos^2 t$ 把根号消去；遇到 $\sqrt{a^2+x^2}$ 设 $x=a\tan t$。换元后别忘了**换回原变量**。

### 分部积分法

由乘积求导法则 $(uv)' = u'v + uv'$ 倒推，得到处理「两类不同函数相乘」的利器：
$$\int u\,dv = uv - \int v\,du.$$

使用关键在于**恰当选 $u$ 和 $dv$**：选 $u$ 要让 $du$ 变简单，选 $dv$ 要容易积出 $v$。经验口诀「反对幂指三」（反三角、对数、幂、指数、三角）给出 $u$ 的优先选取顺序。它特别适合 $\int x e^x dx$、$\int x\sin x\,dx$、$\int \ln x\,dx$ 这类组合；有时连用两次或出现「循环」（如 $\int e^x\sin x\,dx$）后移项解出。

### 定积分与黎曼和

定积分的出发点是「求曲边梯形的面积」。把区间 $[a,b]$ 分成 $n$ 个小段，在每段取一点 $\xi_i$，用一排细长矩形逼近：

**定义（定积分）.**
$$\int_a^b f(x)\,dx = \lim_{\lambda \to 0}\sum_{i=1}^{n} f(\xi_i)\,\Delta x_i,$$
其中 $\Delta x_i$ 是第 $i$ 段的长度，$\lambda$ 是最大段长。这个和式称为**黎曼和**，极限存在时称 $f$ 在 $[a,b]$ 上**可积**。闭区间上连续的函数必可积。

要点：定积分的结果是一个**数**（不带 $C$），它的几何意义是 $f$ 与 $x$ 轴之间的**带符号面积**（$x$ 轴下方计负）。基本性质包括线性性、区间可加性 $\int_a^b=\int_a^c+\int_c^b$，以及保号性（$f\ge 0\Rightarrow\int_a^b f\ge 0$）。

### 微积分基本定理

这条定理把「累加」和「求导」连成一座桥。

**牛顿–莱布尼茨公式.** 若 $f$ 在 $[a,b]$ 上连续，$F$ 是 $f$ 的任一原函数，则
$$\int_a^b f(x)\,dx = F(b) - F(a).$$

它的威力在于：本来要算一个极其复杂的极限（黎曼和），现在只需找出一个原函数、代入两端相减即可。求面积这件「累加」的事，被还原成了「求导的逆运算」。

与之配套的还有**变上限积分求导**：若 $\Phi(x)=\displaystyle\int_a^x f(t)\,dt$，则 $\Phi'(x) = f(x)$——这正说明「积分再求导，还原回自己」，也证明了连续函数必有原函数。定积分换元与分部时，换元要**同时换积分上下限**，分部公式则为 $\int_a^b u\,dv = [uv]_a^b - \int_a^b v\,du$。

### 定积分的应用与反常积分

定积分的本质是「分割、近似、求和、取极限」，凡是能写成这种累加结构的量都能用它计算。

**平面图形面积.** 由 $y=f(x)$、$y=g(x)$（$f\ge g$）及 $x=a,x=b$ 围成的面积：
$$A = \int_a^b [f(x) - g(x)]\,dx.$$

**旋转体体积.** 曲线 $y=f(x)$ 绕 $x$ 轴旋转一周，所得立体的体积（圆盘法）：
$$V = \pi\int_a^b [f(x)]^2\,dx.$$
其原理是：在 $x$ 处切一薄片，近似为半径 $f(x)$、厚 $dx$ 的小圆柱，体积 $\pi[f(x)]^2\,dx$，再沿 $x$ 累加。

**反常积分.** 当积分区间无限（如 $\int_1^{\infty}$）或被积函数在端点无界时，定积分推广为极限：$\displaystyle\int_a^{\infty} f\,dx = \lim_{b\to\infty}\int_a^b f\,dx$。极限存在称**收敛**，否则**发散**。例如 $\int_1^{\infty}\frac{1}{x^p}\,dx$ 当 $p>1$ 收敛、$p\le 1$ 发散。

### 小结

- 原函数：$F'=f$；不定积分 $\int f\,dx=F+C$ 含全部原函数。先背熟基本积分表与线性性。
- 三大方法：凑微分（链式法则逆用）、第二类换元（设 $x=\varphi(t)$ 消根号，记得换回）、分部积分 $\int u\,dv=uv-\int v\,du$（选 $u$ 用「反对幂指三」）。
- 定积分 $\int_a^b f\,dx$ 是黎曼和极限，结果是个数，几何上为带符号面积；连续 ⇒ 可积。
- 牛顿–莱布尼茨公式 $\int_a^b f\,dx=F(b)-F(a)$ 把累加化为求原函数；变上限积分 $\frac{d}{dx}\int_a^x f(t)\,dt=f(x)$。
- 应用：面积 $\int_a^b[f-g]\,dx$，旋转体体积 $\pi\int_a^b f^2\,dx$。
- 反常积分用极限定义；$\int_1^{\infty}x^{-p}\,dx$ 在 $p>1$ 时收敛。

## 典型例题

**例 1.** 求 $\displaystyle\int (3x^2 - 2\cos x)\,dx$。

> **解.** 用线性性质拆开，逐项套基本积分表：
> $$\int (3x^2 - 2\cos x)\,dx = 3\cdot\frac{x^3}{3} - 2\sin x + C = x^3 - 2\sin x + C.$$

**例 2.** 求 $\displaystyle\int x\,e^{x}\,dx$。

> **解.** 这是幂函数乘指数函数，用分部积分。按「反对幂指三」取 $u = x$（求导变简单），$dv = e^x\,dx$，则 $du = dx$、$v = e^x$：
> $$\int x e^x\,dx = x e^x - \int e^x\,dx = x e^x - e^x + C = (x-1)e^x + C.$$

**例 3.** 计算定积分 $\displaystyle\int_0^1 x^2\,dx$，并说明其几何意义。

> **解.** 先求原函数 $F(x)=\dfrac{x^3}{3}$，再用牛顿–莱布尼茨公式：
> $$\int_0^1 x^2\,dx = \left.\frac{x^3}{3}\right|_0^1 = \frac{1^3}{3} - \frac{0^3}{3} = \frac{1}{3}.$$
> 几何上，这就是曲线 $y=x^2$、$x$ 轴与直线 $x=1$ 所围曲边三角形的面积，等于 $\dfrac13$。

**例 4.** 求 $\displaystyle\int_0^{a}\sqrt{a^2 - x^2}\,dx$（$a>0$）。

> **解.** 含 $\sqrt{a^2-x^2}$，用第二类换元（三角代换）。设 $x = a\sin t$，则 $dx = a\cos t\,dt$，且 $\sqrt{a^2-x^2}=a\cos t$（在 $t\in[0,\frac{\pi}{2}]$ 上 $\cos t\ge0$）。换限：$x=0\Rightarrow t=0$，$x=a\Rightarrow t=\frac{\pi}{2}$。于是
> $$\int_0^a\sqrt{a^2-x^2}\,dx = \int_0^{\pi/2} a\cos t\cdot a\cos t\,dt = a^2\int_0^{\pi/2}\cos^2 t\,dt.$$
> 用降幂公式 $\cos^2 t = \dfrac{1+\cos 2t}{2}$：
> $$= a^2\int_0^{\pi/2}\frac{1+\cos 2t}{2}\,dt = \frac{a^2}{2}\left[t + \frac{\sin 2t}{2}\right]_0^{\pi/2} = \frac{a^2}{2}\cdot\frac{\pi}{2} = \frac{\pi a^2}{4}.$$
> 这正是四分之一圆的面积，与几何直观一致。

**例 5.** 求由曲线 $y = \sqrt{x}$、$x$ 轴及 $x=1$ 所围图形绕 $x$ 轴旋转一周所得旋转体的体积。

> **解.** 用圆盘法 $V=\pi\displaystyle\int_a^b [f(x)]^2\,dx$，这里 $f(x)=\sqrt{x}$，区间 $[0,1]$：
> $$V = \pi\int_0^1 (\sqrt{x})^2\,dx = \pi\int_0^1 x\,dx = \pi\left[\frac{x^2}{2}\right]_0^1 = \frac{\pi}{2}.$$

## 练习与自测

1. 求不定积分 $\displaystyle\int \frac{1}{\sqrt{x}}\,dx$。
2. 用凑微分法求 $\displaystyle\int \cos(2x)\,dx$。
3. 求定积分 $\displaystyle\int_0^{\pi} \sin x\,dx$。
4. 求由曲线 $y=x^2$ 与直线 $y=x$ 所围平面图形的面积。
5. 用分部积分求 $\displaystyle\int \ln x\,dx$。
6. 判断反常积分 $\displaystyle\int_1^{\infty}\frac{1}{x^2}\,dx$ 的敛散性，收敛则求其值。

**参考答案.** 1) $\int x^{-1/2}dx = 2x^{1/2}+C = 2\sqrt{x}+C$；2) 令 $u=2x$，$du=2dx$，原式 $=\frac12\sin 2x + C$；3) $\big[-\cos x\big]_0^{\pi} = -\cos\pi-(-\cos 0)= 1+1 = 2$；4) 两曲线交于 $x=0,1$，且在 $(0,1)$ 上 $x\ge x^2$，面积 $=\int_0^1 (x - x^2)\,dx = \frac12 - \frac13 = \frac16$；5) 取 $u=\ln x$、$dv=dx$，则 $du=\frac1x dx$、$v=x$，$\int\ln x\,dx = x\ln x - \int x\cdot\frac1x dx = x\ln x - x + C$；6) $\int_1^{b}x^{-2}dx = \big[-\frac1x\big]_1^b = 1-\frac1b\to 1$（$b\to\infty$），故**收敛**，值为 $1$。
