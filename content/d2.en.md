## Core Notes

### Sets and Operations

A **set** is a collection of distinct, well-defined objects called its **elements**. We write $a\in A$ if $a$ is an element of $A$, and $a\notin A$ otherwise. Standard number sets: $\mathbb{N}$ (naturals), $\mathbb{Z}$ (integers), $\mathbb{Q}$ (rationals), $\mathbb{R}$ (reals). The empty set is $\emptyset$.

If every element of $A$ belongs to $B$, then $A$ is a **subset** of $B$, written $A\subseteq B$; if additionally $A\neq B$ it is a **proper subset** $A\subset B$. All subsets of $A$ form the **power set** $\mathcal{P}(A)$, with $|\mathcal{P}(A)|=2^{|A|}$.

**Set operations** (with universe $U$):

- **Union** $A\cup B=\{x : x\in A \lor x\in B\}$.
- **Intersection** $A\cap B=\{x : x\in A \land x\in B\}$.
- **Difference** $A\setminus B=\{x : x\in A \land x\notin B\}$.
- **Complement** $\overline{A}=U\setminus A$.
- **Cartesian product** $A\times B=\{(a,b) : a\in A \land b\in B\}$, with $|A\times B|=|A|\cdot|B|$.

Useful **identities**: De Morgan $\overline{A\cup B}=\overline{A}\cap\overline{B}$, $\overline{A\cap B}=\overline{A}\cup\overline{B}$; distributivity $A\cap(B\cup C)=(A\cap B)\cup(A\cap C)$; absorption $A\cup(A\cap B)=A$.

### Binary Relations and Properties

A **binary relation** $R$ from $A$ to $B$ is a subset of $A\times B$; we write $(a,b)\in R$ or $a\,R\,b$. When $A=B$, $R$ is a relation on $A$. Three core properties of a relation on $A$:

- **Reflexive**: for all $x\in A$, $x\,R\,x$.
- **Symmetric**: if $x\,R\,y$ then $y\,R\,x$.
- **Transitive**: if $x\,R\,y$ and $y\,R\,z$ then $x\,R\,z$.

A fourth property is **antisymmetry**: if $x\,R\,y$ and $y\,R\,x$ then $x=y$. For example, divisibility $\mid$ on $\mathbb{Z}^+$ is reflexive, antisymmetric, and transitive; so is $\le$ on $\mathbb{R}$.

### Equivalence Relations and Partitions

A relation that is **reflexive, symmetric, and transitive** is an **equivalence relation**, often written $\sim$. The **equivalence class** of $a$ is
$$[a]=\{x\in A : x\sim a\}.$$

A **partition** of $A$ is a family of nonempty, pairwise disjoint subsets whose union is $A$. **Fundamental theorem:** equivalence relations on $A$ correspond bijectively to partitions of $A$ — the equivalence classes of a relation form a partition, and conversely every partition induces a relation (related iff in the same block).

Example: on $\mathbb{Z}$ define $a\sim b \iff a\equiv b\pmod 3$. This yields three classes $[0],[1],[2]$ (by remainder), partitioning $\mathbb{Z}$.

### Partial Orders and Hasse Diagrams

A relation that is **reflexive, antisymmetric, and transitive** is a **partial order**, written $\preceq$; $(A,\preceq)$ is a **poset**. If every two elements are comparable ($x\preceq y$ or $y\preceq x$), it is a **total order**.

A **Hasse diagram** depicts a poset compactly. Drawing rules (in words): draw elements as points with **larger elements higher**; if $x\prec y$ and no $z$ satisfies $x\prec z\prec y$ (i.e. $y$ **covers** $x$), connect $x$ and $y$ with an upward segment; **omit** reflexive loops and any edge implied by transitivity.

Example: the Hasse diagram of $(\{1,2,3,6\},\mid)$ — bottom level $1$; above it $2$ and $3$ (both covering $1$); top level $6$ (covering $2$ and $3$). Since $2$ and $3$ are incomparable, they sit side by side with no edge between them.

### Functions and Cardinality

A **function** $f:A\to B$ is a special relation: each $a\in A$ maps to exactly one $f(a)\in B$. Three important types:

- **Injective (one-to-one)**: $f(x_1)=f(x_2)\implies x_1=x_2$.
- **Surjective (onto)**: every $b\in B$ has some $a$ with $f(a)=b$.
- **Bijective**: both injective and surjective; then an inverse $f^{-1}$ exists.

If a bijection between $A$ and $B$ exists, they are **equinumerous** (same cardinality), written $|A|=|B|$. A set equinumerous with $\mathbb{N}$ (or finite) is **countable**. Both $\mathbb{Z}$ and $\mathbb{Q}$ are countable; $\mathbb{R}$ is **uncountable** (Cantor's diagonal argument), so $|\mathbb{N}|<|\mathbb{R}|$.

## Worked Examples

**Example 1.** Let $A=\{1,2,3\}$, $B=\{2,3,4\}$, universe $U=\{1,2,3,4,5\}$. Find $A\cup B$, $A\cap B$, $A\setminus B$, $\overline{A}$, and the number of elements of $\mathcal{P}(A)$.

> **Solution.** Directly from the definitions:
> $$A\cup B=\{1,2,3,4\},\quad A\cap B=\{2,3\},$$
> $$A\setminus B=\{1\},\quad \overline{A}=U\setminus A=\{4,5\}.$$
> Since $|A|=3$, we get $|\mathcal{P}(A)|=2^3=8$ (including $\emptyset$ and $A$ itself).

**Example 2.** On $\mathbb{Z}$ define $a\,R\,b \iff a-b$ is even. Prove $R$ is an equivalence relation and identify its classes.

> **Solution.** Check each property:
>
> - **Reflexive**: $a-a=0$ is even, so $a\,R\,a$.
> - **Symmetric**: if $a-b$ is even, then $b-a=-(a-b)$ is even, so $b\,R\,a$.
> - **Transitive**: if $a-b$ and $b-c$ are even, then $a-c=(a-b)+(b-c)$ is a sum of evens, hence even, so $a\,R\,c$.
>
> All three hold, so $R$ is an equivalence relation. It has two classes by parity: the evens $[0]$ and the odds $[1]$, which partition $\mathbb{Z}$.

**Example 3.** Decide whether $f:\mathbb{Z}\to\mathbb{Z}$, $f(n)=2n+1$ is injective and/or surjective.

> **Solution.** **Injective**: suppose $f(m)=f(n)$, i.e. $2m+1=2n+1$; subtracting $1$ and dividing by $2$ gives $m=n$, so $f$ is injective.
>
> **Surjective**: $f(n)=2n+1$ is always odd, so the range contains only odd integers. Take $b=2\in\mathbb{Z}$; the equation $2n+1=2$ has no integer solution, so $f$ is **not** surjective.
>
> Conclusion: $f$ is injective but not surjective, hence not bijective.

## Exercises

1. Let $A=\{a,b\}$. List all elements of the power set $\mathcal{P}(A)$ and give $|\mathcal{P}(A)|$.
2. Simplify $\overline{\overline{A}\cup B}$ using De Morgan, expressing the result with intersection/difference of $A,B$.
3. The relation $R=\{(1,1),(2,2),(1,2)\}$ on $\{1,2\}$: determine whether it is reflexive, symmetric, transitive.
4. For the divisibility relation on $\{1,2,3,4,12\}$, list every covering pair in its Hasse diagram (who covers whom).

**Answers.** 1) $\mathcal{P}(A)=\{\emptyset,\{a\},\{b\},\{a,b\}\}$, $|\mathcal{P}(A)|=4$; 2) $\overline{\overline{A}\cup B}=\overline{\overline{A}}\cap\overline{B}=A\cap\overline{B}=A\setminus B$; 3) **Reflexive** (has $(1,1),(2,2)$) and **transitive** (no counterexample), but **not symmetric** ($(1,2)$ present, $(2,1)$ absent); 4) Covering pairs are $1\!\prec\!2$, $1\!\prec\!3$, $2\!\prec\!4$, $4\!\prec\!12$, $3\!\prec\!12$ (note $2\!\prec\!12$ goes through $4$ and $1\!\prec\!4$ through $2$, so neither is a covering pair).
