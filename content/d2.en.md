## Core Notes

### Sets and Operations

A **set** is a collection of distinct, well-defined objects called its **elements**. We write $a\in A$ if $a$ is an element of $A$, and $a\notin A$ otherwise. Standard number sets: $\mathbb{N}$ (naturals), $\mathbb{Z}$ (integers), $\mathbb{Q}$ (rationals), $\mathbb{R}$ (reals). The empty set is $\emptyset$. A set is determined **solely** by its elements, regardless of order or repetition, so $\{1,2\}=\{2,1\}=\{1,1,2\}$.

If every element of $A$ belongs to $B$, then $A$ is a **subset** of $B$, written $A\subseteq B$; if additionally $A\neq B$ it is a **proper subset** $A\subset B$. Two sets are equal, $A=B$, iff $A\subseteq B$ and $B\subseteq A$ — the standard way to prove set equality (**double inclusion**). All subsets of $A$ form the **power set** $\mathcal{P}(A)$, with $|\mathcal{P}(A)|=2^{|A|}$; both $\emptyset$ and $A$ itself are always subsets of $A$.

**Set operations** (with universe $U$):

- **Union** $A\cup B=\{x : x\in A \lor x\in B\}$.
- **Intersection** $A\cap B=\{x : x\in A \land x\in B\}$; if $A\cap B=\emptyset$ the sets are **disjoint**.
- **Difference** $A\setminus B=\{x : x\in A \land x\notin B\}$.
- **Complement** $\overline{A}=U\setminus A$.
- **Cartesian product** $A\times B=\{(a,b) : a\in A \land b\in B\}$, with $|A\times B|=|A|\cdot|B|$. Note $(a,b)$ is an **ordered pair**, so in general $A\times B\neq B\times A$.

**Set-notation cheat sheet:**

| Symbol | Read as / name | Meaning |
|---|---|---|
| $\in$ / $\notin$ | is in / is not in | $a\in A$: $a$ is an element of $A$ |
| $\subseteq$ / $\subset$ | subset / proper subset | $A\subseteq B$: every element of $A$ is in $B$ |
| $\cup$ | union | $A\cup B$: in $A$ or in $B$ |
| $\cap$ | intersection | $A\cap B$: in $A$ and in $B$ |
| $\setminus$ | set difference | $A\setminus B$: in $A$ but not in $B$ |
| $\overline{A}$ | complement | $U\setminus A$: everything in the universe outside $A$ |
| $\emptyset$ | empty set | the set with no elements |
| $\times$ | Cartesian product | $A\times B$: all ordered pairs $(a,b)$ |
| $\lvert A\rvert$ | cardinality | the number of elements of $A$ |
| $\mathcal{P}(A)$ | power set | the set of all subsets of $A$ |

(For the number sets $\mathbb{N},\mathbb{Z},\mathbb{Q},\mathbb{R}$ see F1 "Numbers & Expressions", ch. 1; for the connectives $\lor,\land,\neg$ see "Mathematical Logic" in this module.)

Useful **identities** (mirroring the logical equivalences, since $\cup,\cap,\overline{\,\cdot\,}$ correspond to $\lor,\land,\neg$): De Morgan $\overline{A\cup B}=\overline{A}\cap\overline{B}$, $\overline{A\cap B}=\overline{A}\cup\overline{B}$; distributivity $A\cap(B\cup C)=(A\cap B)\cup(A\cap C)$; absorption $A\cup(A\cap B)=A$. Each can be proved by **double inclusion** or via the logical equivalence of element membership.

### Binary Relations and Properties

A **binary relation** $R$ from $A$ to $B$ is a subset of $A\times B$; we write $(a,b)\in R$ or $a\,R\,b$. When $A=B$, $R$ is a relation on $A$. If $|A|=n$, there are $2^{n^2}$ distinct relations on $A$ (since $|A\times A|=n^2$ and each ordered pair is in or out of $R$). Core properties of a relation on $A$:

- **Reflexive**: for all $x\in A$, $x\,R\,x$.
- **Irreflexive**: for all $x$, $(x,x)\notin R$.
- **Symmetric**: if $x\,R\,y$ then $y\,R\,x$.
- **Antisymmetric**: if $x\,R\,y$ and $y\,R\,x$ then $x=y$.
- **Transitive**: if $x\,R\,y$ and $y\,R\,z$ then $x\,R\,z$.

Note that "symmetric" and "antisymmetric" are not mutually exclusive: equality $=$ is both; but in general they are independent. For example, divisibility $\mid$ on $\mathbb{Z}^+$ is reflexive, antisymmetric, transitive; so is $\le$ on $\mathbb{R}$; while $<$ is irreflexive, antisymmetric, transitive.

A handy test: view a relation on $A$ as a directed graph (vertices = elements, draw an arc $a\to b$ when $a\,R\,b$). Then reflexive = a loop at every vertex; symmetric = arcs come in pairs; transitive = every "two-step path" has a "one-step shortcut".

### Equivalence Relations and Partitions

A relation that is **reflexive, symmetric, and transitive** is an **equivalence relation**, often written $\sim$. The **equivalence class** of $a$ is
$$[a]=\{x\in A : x\sim a\}.$$
Equivalence classes have two key properties: each $a$ lies in its own class ($a\in[a]$); and two classes are either **identical** or **disjoint** — that is, $[a]=[b]\iff a\sim b$, otherwise $[a]\cap[b]=\emptyset$.

A **partition** of $A$ is a family of nonempty, pairwise disjoint subsets (the **blocks**) whose union is $A$. **Fundamental theorem:** equivalence relations on $A$ correspond bijectively to partitions of $A$ — the equivalence classes of a relation form a partition, and conversely every partition induces a relation (related iff in the same block).

Example: on $\mathbb{Z}$ define $a\sim b \iff a\equiv b\pmod 3$. This yields three classes $[0],[1],[2]$ (by remainder), partitioning $\mathbb{Z}$. These classes, written $\mathbb{Z}/3\mathbb{Z}$ or $\mathbb{Z}_3$, are the basis of modular arithmetic.

### Partial Orders and Hasse Diagrams

A relation that is **reflexive, antisymmetric, and transitive** is a **partial order**, written $\preceq$; $(A,\preceq)$ is a **poset**. If every two elements are comparable ($x\preceq y$ or $y\preceq x$), it is a **total order** (a chain); if no two are comparable, it is an **antichain**.

A **Hasse diagram** depicts a poset compactly. Drawing rules (in words): draw elements as points with **larger elements higher**; if $x\prec y$ and no $z$ satisfies $x\prec z\prec y$ (i.e. $y$ **covers** $x$), connect $x$ and $y$ with an upward segment; **omit** reflexive loops and any edge implied by transitivity.

Example: the Hasse diagram of $(\{1,2,3,6\},\mid)$ — bottom level $1$; above it $2$ and $3$ (both covering $1$); top level $6$ (covering $2$ and $3$). Since $2$ and $3$ are incomparable, they sit side by side with no edge between them.

**Special elements** of a poset: $m$ is **maximal** if no $x$ satisfies $m\prec x$ (sits "at the top" of the diagram, possibly non-unique); $M$ is the **greatest** element if $x\preceq M$ for all $x$ (unique if it exists). Minimal and least elements are defined dually. A greatest element is always maximal, but not conversely.

### Functions and Cardinality

A **function** $f:A\to B$ is a special relation: each $a\in A$ maps to exactly one $f(a)\in B$. Here $A$ is the domain, $B$ the codomain, and $\{f(a):a\in A\}$ the range. Three important types:

- **Injective (one-to-one)**: $f(x_1)=f(x_2)\implies x_1=x_2$ (distinct inputs give distinct outputs).
- **Surjective (onto)**: every $b\in B$ has some $a$ with $f(a)=b$ (range equals codomain).
- **Bijective**: both injective and surjective; then an inverse $f^{-1}$ exists.

Composition $g\circ f$ is defined by $(g\circ f)(x)=g(f(x))$. If $A,B$ are finite with $|A|=|B|$, then $f:A\to B$ is **injective $\iff$ surjective $\iff$ bijective** (a pigeonhole consequence); this fails for infinite sets (e.g. $f(n)=2n$ on $\mathbb{N}$ is injective but not surjective).

If a bijection between $A$ and $B$ exists, they are **equinumerous** (same cardinality), written $|A|=|B|$. A set equinumerous with $\mathbb{N}$ (or finite) is **countable**. Both $\mathbb{Z}$ and $\mathbb{Q}$ are countable; $\mathbb{R}$ is **uncountable** (Cantor's diagonal argument), so $|\mathbb{N}|<|\mathbb{R}|$. Thus even infinities come in different sizes.

### Summary

- A set is determined by its elements; $A=B\iff A\subseteq B\land B\subseteq A$; $|\mathcal{P}(A)|=2^{|A|}$.
- Set identities mirror logical equivalences: $\cup\leftrightarrow\lor$, $\cap\leftrightarrow\land$, $\overline{\cdot}\leftrightarrow\neg$.
- Five properties: reflexive, irreflexive, symmetric, antisymmetric, transitive; symmetric and antisymmetric are not mutually exclusive.
- Equivalence relation $\iff$ partition; classes are pairwise either equal or disjoint.
- Partial order = reflexive + antisymmetric + transitive; a Hasse diagram shows only covers; distinguish maximal from greatest.
- Three function types: injective, surjective, bijective; equivalent on equal-size finite sets but not on infinite sets; $\mathbb{Z},\mathbb{Q}$ countable, $\mathbb{R}$ uncountable.

## Worked Examples

**Example 1.** Let $A=\{1,2,3\}$, $B=\{2,3,4\}$, universe $U=\{1,2,3,4,5\}$. Find $A\cup B$, $A\cap B$, $A\setminus B$, $\overline{A}$, and the number of elements of $\mathcal{P}(A)$.

> **Solution.** Directly from the definitions:
> $$A\cup B=\{1,2,3,4\},\quad A\cap B=\{2,3\},$$
> $$A\setminus B=\{1\},\quad \overline{A}=U\setminus A=\{4,5\}.$$
> Since $|A|=3$, we get $|\mathcal{P}(A)|=2^3=8$ (including $\emptyset$ and $A$ itself).

**Example 2.** On $\mathbb{Z}$ define $a\,R\,b \iff a-b$ is even. Prove $R$ is an equivalence relation and identify its classes.

> **Solution.** Check each property:
>
> - **Reflexive**: $a-a=0$ is even, so $a\,R\,a$.
> - **Symmetric**: if $a-b$ is even, then $b-a=-(a-b)$ is even, so $b\,R\,a$.
> - **Transitive**: if $a-b$ and $b-c$ are even, then $a-c=(a-b)+(b-c)$ is a sum of evens, hence even, so $a\,R\,c$.
>
> All three hold, so $R$ is an equivalence relation. It has two classes by parity: the evens $[0]$ and the odds $[1]$, which partition $\mathbb{Z}$.

**Example 3.** Decide whether $f:\mathbb{Z}\to\mathbb{Z}$, $f(n)=2n+1$ is injective and/or surjective.

> **Solution.** **Injective**: suppose $f(m)=f(n)$, i.e. $2m+1=2n+1$; subtracting $1$ and dividing by $2$ gives $m=n$, so $f$ is injective.
>
> **Surjective**: $f(n)=2n+1$ is always odd, so the range contains only odd integers. Take $b=2\in\mathbb{Z}$; the equation $2n+1=2$ has no integer solution, so $f$ is **not** surjective.
>
> Conclusion: $f$ is injective but not surjective, hence not bijective.

**Example 4.** Prove the distributive law $A\cap(B\cup C)=(A\cap B)\cup(A\cap C)$ by double inclusion.

> **Solution.** Prove both inclusions. Take an arbitrary $x$:
>
> "$\subseteq$": if $x\in A\cap(B\cup C)$, then $x\in A$ and $x\in B\cup C$, i.e. $x\in B$ or $x\in C$. If $x\in B$, then $x\in A\cap B$; if $x\in C$, then $x\in A\cap C$. Either way $x\in(A\cap B)\cup(A\cap C)$.
>
> "$\supseteq$": if $x\in(A\cap B)\cup(A\cap C)$, then $x\in A\cap B$ or $x\in A\cap C$. In both cases $x\in A$, and $x\in B$ or $x\in C$, i.e. $x\in B\cup C$, so $x\in A\cap(B\cup C)$.
>
> Both inclusions hold, so the sets are equal. At heart this step is the logical distributive law $p\land(q\lor r)\equiv(p\land q)\lor(p\land r)$.

**Example 5.** Let $A=\{1,2,3,4,6,12\}$ with the divisibility relation $\mid$. Verify it is a partial order, list the covering pairs of its Hasse diagram, and identify the greatest and least elements.

> **Solution.** Divisibility on positive integers is reflexive ($a\mid a$), antisymmetric ($a\mid b$ and $b\mid a\Rightarrow a=b$), and transitive ($a\mid b$ and $b\mid c\Rightarrow a\mid c$), so $(A,\mid)$ is a poset.
>
> Covering pairs (all $x\prec y$ with no element strictly between): $1\!\prec\!2$, $1\!\prec\!3$, $2\!\prec\!4$, $2\!\prec\!6$, $3\!\prec\!6$, $4\!\prec\!12$, $6\!\prec\!12$. For instance $1\!\prec\!4$ is not a cover because it passes through $2$.
>
> The **least** element is $1$ (it divides everything), and the **greatest** is $12$ (everything divides it). Both exist and are unique, so this poset has both a bottom and a top.

## Exercises

1. Let $A=\{a,b\}$. List all elements of the power set $\mathcal{P}(A)$ and give $|\mathcal{P}(A)|$.
2. Simplify $\overline{\overline{A}\cup B}$ using De Morgan, expressing the result with intersection/difference of $A,B$.
3. The relation $R=\{(1,1),(2,2),(1,2)\}$ on $\{1,2\}$: determine whether it is reflexive, symmetric, antisymmetric, transitive.
4. For the divisibility relation on $\{1,2,3,4,12\}$, list every covering pair in its Hasse diagram (who covers whom).
5. Let $|A|=3$, $|B|=4$. Find $|A\times B|$ and the total number of functions from $A$ to $B$.
6. Decide whether $f:\mathbb{R}\to\mathbb{R}$, $f(x)=3x-5$ is a bijection; if so, find $f^{-1}$.

**Answers.** 1) $\mathcal{P}(A)=\{\emptyset,\{a\},\{b\},\{a,b\}\}$, $|\mathcal{P}(A)|=4$; 2) $\overline{\overline{A}\cup B}=\overline{\overline{A}}\cap\overline{B}=A\cap\overline{B}=A\setminus B$; 3) **reflexive** (has $(1,1),(2,2)$), **antisymmetric** (no $a\neq b$ with both directions), **transitive** (no counterexample), but **not symmetric** ($(1,2)$ present, $(2,1)$ absent); 4) covering pairs are $1\!\prec\!2$, $1\!\prec\!3$, $2\!\prec\!4$, $4\!\prec\!12$, $3\!\prec\!12$ (note $2\!\prec\!12$ goes through $4$ and $1\!\prec\!4$ through $2$, so neither is a covering pair); 5) $|A\times B|=3\times4=12$; the number of functions is $|B|^{|A|}=4^3=64$ (each $a$ independently picks one image); 6) **bijection**: injective since $3x_1-5=3x_2-5\Rightarrow x_1=x_2$, surjective since any $y$ is hit by $x=(y+5)/3$; the inverse is $f^{-1}(y)=\dfrac{y+5}{3}$.
