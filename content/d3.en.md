## Core Notes

### Basic Concepts

An (undirected) **graph** is a pair $G=(V,E)$, where $V$ is a nonempty set of **vertices** and $E$ a set of **edges**, each edge joining a pair of vertices. A graph with at most one edge per vertex pair and no self-loops is a **simple graph**.

The **degree** $\deg(v)$ of a vertex is the number of edges incident to it (a loop counts twice). **Handshaking theorem:**
$$\sum_{v\in V}\deg(v)=2|E|.$$
Intuitively, each edge contributes one to the degree of each of its two endpoints. **Corollary:** the number of vertices of odd degree is even.

A simple graph on $n$ vertices in which every pair is adjacent is the **complete graph** $K_n$, with $\binom{n}{2}=\dfrac{n(n-1)}{2}$ edges. If the vertices split into two parts with edges only between parts, the graph is **bipartite**.

### Matrix Representations

Let $V=\{v_1,\dots,v_n\}$. The **adjacency matrix** $A=(a_{ij})$ is an $n\times n$ matrix: in a simple graph $a_{ij}=1$ when $v_i$ and $v_j$ are adjacent, else $a_{ij}=0$. For an undirected graph $A$ is **symmetric**, and the sum of row $i$ (or column $i$) equals $\deg(v_i)$.

**Meaning of powers:** the $(i,j)$ entry of $A^k$ counts the walks of **length exactly $k$** from $v_i$ to $v_j$. For instance, the diagonal entry $(i,i)$ of $A^2$ gives the number of closed walks of length $2$ through $v_i$, which equals $\deg(v_i)$ in a simple graph.

Example: the triangle $K_3$ on $v_1,v_2,v_3$ has adjacency matrix
$$A=\begin{pmatrix}0&1&1\\1&0&1\\1&1&0\end{pmatrix}.$$
Each row sums to $2$, matching every vertex having degree $2$.

### Walks, Circuits and Connectivity

An alternating sequence of vertices and edges $v_0,e_1,v_1,\dots,e_k,v_k$ is a **walk** of length $k$. A walk with distinct edges is a **trail**; one with distinct vertices is a **path**. A walk with equal start and end is **closed**; a closed trail is a **circuit**; a circuit with distinct vertices (except endpoints) is a **cycle**.

If every two vertices are joined by some walk, $G$ is **connected**. A general graph decomposes into maximal connected subgraphs called **connected components**.

### Euler and Hamiltonian Graphs

A circuit traversing **every edge exactly once** is an **Euler circuit**; a connected graph having one is an **Eulerian graph**. If only such a trail (not necessarily closed) is required, it is an **Euler trail**.

**Euler's theorem (connected undirected graph):**

- An **Euler circuit** exists $\iff$ every vertex has even degree.
- An **Euler trail** exists $\iff$ there are exactly $0$ or $2$ vertices of odd degree (when $2$, the trail must start and end at them).

A cycle visiting **every vertex exactly once** is a **Hamiltonian cycle**; a graph having one is **Hamiltonian**. Unlike the Euler case, there is **no simple characterization** for Hamiltonicity (it is NP-complete). A common **sufficient** condition is Dirac's theorem: in a simple graph with $n\ge3$, if every vertex has $\deg(v)\ge n/2$, then $G$ has a Hamiltonian cycle.

### Trees and Spanning Trees

A **tree** is a connected acyclic graph; an acyclic graph not required to be connected is a **forest**. A tree on $n$ vertices has exactly $n-1$ edges, and there is a **unique** path between any two vertices. The following statements about an $n$-vertex graph $T$ are **mutually equivalent** (equivalent definitions of a tree):

1. $T$ is connected and acyclic;
2. $T$ is connected with exactly $n-1$ edges;
3. $T$ is acyclic with exactly $n-1$ edges;
4. there is a unique path between any two vertices of $T$;
5. $T$ is connected, but removing any edge disconnects it (edge-minimal connected).

A **spanning tree** of a connected graph $G$ is a tree subgraph containing all vertices of $G$. On a weighted graph, the spanning tree of least total weight is a **minimum spanning tree**, found by **Kruskal** (add edges by increasing weight, avoiding cycles) or **Prim** (grow from one vertex).

## Worked Examples

**Example 1.** A graph has $6$ vertices with degrees $4,3,3,2,2,2$. Find the number of edges and decide whether this degree sequence is possible.

> **Solution.** By handshaking, $\sum\deg(v)=4+3+3+2+2+2=16=2|E|$, so $|E|=8$. The odd-degree vertices (the two of degree $3$) number $2$, which is even, consistent with the corollary; hence the **sequence is feasible**.

**Example 2.** Is the complete graph $K_4$ Eulerian? What about $K_5$?

> **Solution.** In $K_n$ every vertex has degree $n-1$.
>
> - $K_4$: each degree $=3$, odd, so odd-degree vertices exist (in fact $4$ of them). By Euler's theorem it is **not Eulerian**, and has no Euler trail either (more than $2$ odd vertices).
> - $K_5$: each degree $=4$, all even, and $K_5$ is connected, so it **is Eulerian** with an Euler circuit.

**Example 3.** A tree has $10$ vertices — how many edges? And: if a connected graph on $10$ vertices has exactly $9$ edges, must it be a tree?

> **Solution.** A tree on $n$ vertices has exactly $n-1$ edges, so a $10$-vertex tree has $\mathbf{9}$ edges.
>
> Conversely, by the equivalent definition (statement 2), "connected + exactly $n-1$ edges" forces acyclicity, hence a tree. The given graph is connected on $10$ vertices with exactly $9=n-1$ edges, so it **must be a tree**.

## Exercises

1. A simple graph has $7$ vertices, each of degree $4$. Find the number of edges.
2. Write the adjacency matrix of the path graph $P_3$ (vertices $v_1\!-\!v_2\!-\!v_3$, two edges) and verify each row sum equals the degree of the corresponding vertex.
3. A connected graph has $4$ vertices of odd degree. Does it have an Euler trail? An Euler circuit? Justify.
4. How many edges does $K_5$ have? How many edges does a spanning tree of it have?

**Answers.** 1) $\sum\deg=7\times4=28=2|E|$, so $|E|=14$; 2) $A=\begin{pmatrix}0&1&0\\1&0&1\\0&1&0\end{pmatrix}$, row sums $1,2,1$, exactly $\deg(v_1)=1,\deg(v_2)=2,\deg(v_3)=1$; 3) with $4$ odd-degree vertices ($>2$), by Euler's theorem there is **neither an Euler trail nor an Euler circuit**; 4) $K_5$ has $\binom{5}{2}=10$ edges, and its spanning tree (on $5$ vertices) has $5-1=4$ edges.
