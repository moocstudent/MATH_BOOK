## 核心讲义

如果极限回答的是「无限接近会怎样」，导数回答的就是「变化有多快」。它把瞬时变化率、曲线的切线斜率、函数的局部线性化这三件看似不同的事统一成一个概念，并由此衍生出研究函数升降、凹凸、极值的一整套微分学工具。

### 导数的定义与几何意义

**定义（导数）.** 设 $y=f(x)$ 在 $x_0$ 的邻域内有定义。若极限
$$f'(x_0) = \lim_{\Delta x \to 0}\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$$
存在，则称 $f$ 在 $x_0$ **可导**，该极限值即为 $f$ 在 $x_0$ 的**导数**，也记作 $\dfrac{dy}{dx}\Big|_{x_0}$ 或 $y'(x_0)$。

分式 $\dfrac{f(x_0+\Delta x)-f(x_0)}{\Delta x}$ 是**平均变化率**，即割线斜率；令 $\Delta x\to 0$ 后割线趋于切线。换一个等价写法（令 $x=x_0+\Delta x$）：$f'(x_0)=\lim_{x\to x_0}\dfrac{f(x)-f(x_0)}{x-x_0}$。

**几何意义.** $f'(x_0)$ 就是曲线 $y=f(x)$ 在点 $(x_0, f(x_0))$ 处**切线的斜率**。切线方程为
$$y - f(x_0) = f'(x_0)\,(x - x_0),$$
与之垂直的**法线**斜率为 $-\dfrac{1}{f'(x_0)}$（当 $f'(x_0)\ne 0$）。

**可导与连续的关系.** 可导必连续；但连续不一定可导，典型反例是 $f(x)=|x|$ 在 $x=0$（有「尖点」，左导数 $-1$、右导数 $+1$，不相等）。可见可导是比连续**更强**的光滑性要求：图像不仅不断开，还要有唯一确定的切线方向。

### 求导法则：四则与链式

逐项用定义求导太慢，实际计算依赖法则。设 $u,v$ 可导，$C$ 为常数：

$$(Cu)' = Cu', \qquad (u\pm v)' = u' \pm v',$$
$$(uv)' = u'v + uv', \qquad \left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}\ (v\ne 0).$$

注意乘积法则不是 $u'v'$——这是最常见的错误。乘积法则可由定义证明：把分子 $u(x+\Delta x)v(x+\Delta x)-u(x)v(x)$ 加减一项 $u(x+\Delta x)v(x)$ 拆成两块，分别取极限即得。

**链式法则（复合函数求导）.** 这是求导中最重要的工具。若 $y = f(u)$、$u = g(x)$，则
$$\frac{dy}{dx} = \frac{dy}{du}\cdot\frac{du}{dx} = f'(g(x))\cdot g'(x).$$
口诀：「由外向内，逐层相乘」。多层复合就连乘多个因子，是处理 $\sin(x^2)$、$e^{\cos x}$、$\ln\sqrt{1+x^2}$ 这类表达式的统一方法。

常用基本导数（务必背熟）：
$$(x^a)' = a x^{a-1},\quad (e^x)' = e^x,\quad (a^x)'=a^x\ln a,\quad (\ln x)' = \frac{1}{x},\quad (\sin x)' = \cos x,\quad (\cos x)' = -\sin x,$$
$$(\tan x)' = \frac{1}{\cos^2 x} = \sec^2 x,\qquad (\arcsin x)' = \frac{1}{\sqrt{1-x^2}},\qquad (\arctan x)' = \frac{1}{1 + x^2}.$$

### 隐函数与参数方程求导

**隐函数求导.** 当 $y$ 由方程 $F(x,y)=0$ 隐式确定、难以解出显式表达式时，直接对方程两边关于 $x$ 求导，把 $y$ 视为 $x$ 的函数（即对含 $y$ 的项要乘 $\dfrac{dy}{dx}$），再解出 $\dfrac{dy}{dx}$。

例如对 $x^2 + y^2 = R^2$ 两边求导：$2x + 2y\,y' = 0$，得 $y' = -\dfrac{x}{y}$。对于幂指函数 $y=u(x)^{v(x)}$，常用**对数求导法**：两边取 $\ln$ 化乘除为加减、化幂为乘，再求导，能大幅简化。

**参数方程求导.** 若曲线由 $\begin{cases} x = \varphi(t) \\ y = \psi(t)\end{cases}$ 给出，则
$$\frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{\psi'(t)}{\varphi'(t)}\quad (\varphi'(t)\ne 0).$$

### 微分中值定理与高阶导数

**拉格朗日中值定理.** 若 $f$ 在 $[a,b]$ 连续、在 $(a,b)$ 可导，则存在 $\xi\in(a,b)$ 使
$$f(b) - f(a) = f'(\xi)\,(b - a).$$
它把「整体的平均变化率」与「某点的瞬时变化率」联系起来，是「由导数符号判断函数升降」的理论根据：$f'>0$ 区间上 $f$ 严格递增，$f'<0$ 区间上严格递减。其特例 $f(a)=f(b)$ 即**罗尔定理**。

把导数 $f'(x)$ 再求导，得到**二阶导数** $f''(x)=\dfrac{d^2 y}{dx^2}$；继续下去得到 $n$ 阶导数 $f^{(n)}(x)$。二阶导数有清晰的物理与几何含义：若 $s(t)$ 是位移，则 $s'(t)$ 是速度、$s''(t)$ 是加速度；在几何上，$f''>0$ 对应曲线**下凸（凹向上）**，$f''<0$ 对应**上凸（凹向下）**，凹凸变号处是**拐点**。

几个常见的高阶导数公式：
$$(e^x)^{(n)} = e^x, \qquad (\sin x)^{(n)} = \sin\left(x + \frac{n\pi}{2}\right).$$

### 微分与线性近似

**定义（微分）.** 若 $f$ 在 $x_0$ 可导，则函数增量可写成
$$\Delta y = f'(x_0)\,\Delta x + o(\Delta x),$$
其线性主部 $f'(x_0)\,\Delta x$ 称为 $f$ 在 $x_0$ 的**微分**，记作 $dy = f'(x_0)\,dx$。

微分的核心思想是**用直代曲**：当 $\Delta x$ 很小时，用切线上的增量 $dy$ 近似真实增量 $\Delta y$。由此得到**线性近似公式**：
$$f(x_0 + \Delta x) \approx f(x_0) + f'(x_0)\,\Delta x.$$

它是后续泰勒展开的一阶版本，也是工程中估算、误差分析的常用手段（例如由测量误差 $\Delta x$ 估计函数值的误差 $|dy|=|f'(x_0)|\,|\Delta x|$）。

### 小结

- 导数即差商的极限 $f'(x_0)=\lim_{\Delta x\to0}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}$；几何上是切线斜率。**可导 ⇒ 连续**，反之不真。
- 四则法则：$(uv)'=u'v+uv'$，$\left(\frac uv\right)'=\frac{u'v-uv'}{v^2}$；乘积法则**不是** $u'v'$。
- 链式法则：$\frac{dy}{dx}=\frac{dy}{du}\cdot\frac{du}{dx}$，由外向内逐层相乘。
- 隐函数：两边对 $x$ 求导，含 $y$ 项乘 $y'$ 再解出；参数式 $\frac{dy}{dx}=\frac{\psi'(t)}{\varphi'(t)}$。
- 拉格朗日中值定理 $f(b)-f(a)=f'(\xi)(b-a)$；$f'>0$ 递增、$f'<0$ 递减，$f''>0$ 凹向上、$f''<0$ 凹向下。
- 微分 $dy=f'(x_0)\,dx$，线性近似 $f(x_0+\Delta x)\approx f(x_0)+f'(x_0)\Delta x$。

## 典型例题

**例 1.** 求 $y = x^2 \sin x$ 的导数。

> **解.** 这是两个函数的乘积，用乘积法则 $(uv)'=u'v+uv'$，取 $u=x^2$、$v=\sin x$：
> $$y' = (x^2)'\sin x + x^2(\sin x)' = 2x\sin x + x^2\cos x.$$

**例 2.** 求 $y = \ln(\cos x)$ 的导数。

> **解.** 这是复合函数：外层 $\ln u$，内层 $u=\cos x$。用链式法则，$\dfrac{dy}{du}=\dfrac1u$，$\dfrac{du}{dx}=-\sin x$：
> $$y' = \frac{1}{\cos x}\cdot(-\sin x) = -\frac{\sin x}{\cos x} = -\tan x.$$

**例 3.** 用微分估算 $\sqrt{4.02}$ 的近似值。

> **解.** 取 $f(x)=\sqrt{x}$，则 $f'(x)=\dfrac{1}{2\sqrt{x}}$。选取容易计算的 $x_0=4$，增量 $\Delta x = 0.02$。由线性近似公式：
> $$f(4.02) \approx f(4) + f'(4)\,\Delta x = 2 + \frac{1}{2\cdot 2}\times 0.02 = 2 + 0.005 = 2.005.$$
> （真值约 $2.004994$，误差极小，可见线性近似在小增量下相当准确。）

**例 4.** 求幂指函数 $y = x^{x}$（$x>0$）的导数。

> **解.** 底和指数都含 $x$，既非幂函数也非指数函数，用**对数求导法**。两边取自然对数：
> $$\ln y = x\ln x.$$
> 两边对 $x$ 求导，左边把 $y$ 看成 $x$ 的函数（乘 $\dfrac{y'}{y}$），右边用乘积法则：
> $$\frac{y'}{y} = 1\cdot\ln x + x\cdot\frac1x = \ln x + 1.$$
> 解出 $y'$ 并代回 $y=x^x$：
> $$y' = y\,(\ln x + 1) = x^{x}\,(\ln x + 1).$$

**例 5.** 求函数 $f(x) = x^3 - 3x$ 的单调区间与极值。

> **解.** 这是微分应用：用一阶导数符号判断升降。求导
> $$f'(x) = 3x^2 - 3 = 3(x-1)(x+1).$$
> 令 $f'(x)=0$ 得驻点 $x=-1,\ 1$，它们把数轴分成三段：
> - $x<-1$：$f'>0$，$f$ 递增；
> - $-1<x<1$：$f'<0$，$f$ 递减；
> - $x>1$：$f'>0$，$f$ 递增。
>
> 于是 $f$ 在 $x=-1$ 由增转减，取**极大值** $f(-1)=-1+3=2$；在 $x=1$ 由减转增，取**极小值** $f(1)=1-3=-2$。

## 练习与自测

1. 用导数定义求 $f(x)=x^2$ 在 $x=1$ 处的导数 $f'(1)$。
2. 求 $y = \dfrac{x}{1+x^2}$ 的导数。
3. 设 $x^3 + y^3 = 6xy$，用隐函数求导法求 $\dfrac{dy}{dx}$。
4. 求曲线 $y = e^x$ 在点 $(0,1)$ 处的切线方程。
5. 求 $y = \arctan(x^2)$ 的导数。
6. 求 $f(x) = 2x^3 - 6x^2 + 1$ 的单调区间。

**参考答案.** 1) $\frac{(1+\Delta x)^2-1}{\Delta x}=2+\Delta x\to 2$，故 $f'(1)=2$；2) 用商法则，$y'=\dfrac{(1+x^2)-x\cdot 2x}{(1+x^2)^2}=\dfrac{1-x^2}{(1+x^2)^2}$；3) 两边求导得 $3x^2+3y^2 y'=6y+6x y'$，解出 $y'=\dfrac{2y-x^2}{y^2-2x}$；4) $y'=e^x$，在 $x=0$ 处斜率为 $1$，切线方程 $y = x + 1$；5) 链式法则，$y'=\dfrac{1}{1+(x^2)^2}\cdot 2x=\dfrac{2x}{1+x^4}$；6) $f'(x)=6x^2-12x=6x(x-2)$，故在 $(-\infty,0)$ 与 $(2,+\infty)$ 上递增、在 $(0,2)$ 上递减。
