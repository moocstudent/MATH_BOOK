## Core Notes

Infinite series ask whether "adding up infinitely many numbers" means anything, and if so, what it equals. The ultimate goal is to expand complicated functions as power series (for instance $e^x = 1 + x + \frac{x^2}{2!}+\cdots$) so that any function can be approximated, computed, and analyzed with polynomials — a core tool of numerical computation and modern analysis.

### Numerical series and convergence

Given a sequence $\{a_n\}$, the formal sum $\displaystyle\sum_{n=1}^{\infty} a_n = a_1 + a_2 + a_3 + \cdots$ is a **numerical series**. How do we give it a precise "sum"? Through its **sequence of partial sums**.

**Definition.** Let $S_n = a_1 + a_2 + \cdots + a_n$ be the $n$-th partial sum. If $\lim_{n\to\infty} S_n = S$ exists, the series **converges** with sum $S$; if the limit does not exist, the series **diverges**.

The most important template is the **geometric series**:
$$\sum_{n=0}^{\infty} a r^{n} = \frac{a}{1 - r}\quad (|r| < 1),\qquad \text{and it diverges for }|r|\ge 1.$$

**Necessary condition for convergence.** If $\sum a_n$ converges, then $\lim_{n\to\infty} a_n = 0$. The contrapositive is handy: if $a_n$ does not tend to $0$, the series must diverge. The converse **fails**, however — terms tending to zero do not guarantee convergence. The classic counterexample is the **harmonic series** $\sum\frac1n$, which diverges.

### Tests for positive series

If every $a_n \ge 0$, the series is a **positive series**. Then the partial sums $S_n$ are increasing, so "convergence holds if and only if $S_n$ is bounded" — and from this grows a practical toolkit of tests.

**Comparison test.** Suppose $0 \le a_n \le b_n$: if $\sum b_n$ converges then $\sum a_n$ converges; if $\sum a_n$ diverges then $\sum b_n$ diverges. The standard benchmark is the **p-series**:
$$\sum_{n=1}^{\infty} \frac{1}{n^p}\quad \begin{cases} \text{converges}, & p > 1, \\[2pt] \text{diverges}, & p \le 1.\end{cases}$$

**Ratio test (d'Alembert).** Let $\displaystyle\lim_{n\to\infty}\frac{a_{n+1}}{a_n} = \rho$: it converges if $\rho < 1$, diverges if $\rho > 1$, and is inconclusive if $\rho = 1$. This is the first choice for series with a factorial $n!$ or an exponential $r^n$.

**Root test (Cauchy).** Let $\displaystyle\lim_{n\to\infty}\sqrt[n]{a_n} = \rho$, with the same conclusions. It is especially convenient when the term has the form $(\cdots)^n$.

### Alternating series and absolute convergence

A series whose terms alternate in sign, $\displaystyle\sum (-1)^{n-1} u_n$ with $u_n>0$, is an **alternating series**.

**Leibniz test.** If $\{u_n\}$ is decreasing and $\lim_{n\to\infty} u_n = 0$, then the alternating series converges. For example, the **alternating harmonic series** $\sum\frac{(-1)^{n-1}}{n}$ converges (to $\ln 2$).

For series of arbitrary sign, introduce a stronger notion:

**Absolute vs. conditional convergence.** If $\sum |a_n|$ converges, then $\sum a_n$ **converges absolutely**; if $\sum a_n$ converges but $\sum|a_n|$ diverges, it **converges conditionally**. The key fact: **absolute convergence implies convergence**. So one usually starts by applying the positive-series tests to $|a_n|$ — the standard opening move for a general series. The alternating harmonic series is the model of "conditional but not absolute" convergence (since $\sum\frac1n$ diverges).

### Power series and radius of convergence

A series of the form
$$\sum_{n=0}^{\infty} a_n (x - x_0)^{n} = a_0 + a_1(x-x_0) + a_2(x-x_0)^2 + \cdots$$
is a **power series**, and its sum is a function of $x$. The central question is: for which $x$ does it converge?

**Theorem (Abel).** Every power series has a **radius of convergence** $R$ such that it converges absolutely for $|x - x_0| < R$ and diverges for $|x-x_0|>R$ (the endpoints $|x-x_0|=R$ must be checked separately). The set of all $x$ where it converges is the **interval of convergence**.

The radius is usually found by the ratio method:
$$R = \lim_{n\to\infty}\left|\frac{a_n}{a_{n+1}}\right|.$$
Within the interval of convergence a power series may be **differentiated and integrated term by term**, with the radius unchanged — which makes it an ideal tool for representing and manipulating functions.

### Taylor series and applications

**Taylor series.** If $f$ has derivatives of all orders at $x_0$, it can be expanded near $x_0$ as
$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(x_0)}{n!}(x - x_0)^{n}.$$
The special case $x_0 = 0$ is the **Maclaurin series**. The idea is "approximate a function arbitrarily well by a polynomial", with the $n$-th coefficient set precisely by the $n$-th derivative of $f$ at $x_0$.

A few expansions worth memorizing (each valid on its own domain):
$$e^{x} = \sum_{n=0}^{\infty} \frac{x^{n}}{n!} = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \cdots,$$
$$\sin x = \sum_{n=0}^{\infty} \frac{(-1)^{n} x^{2n+1}}{(2n+1)!}, \qquad \cos x = \sum_{n=0}^{\infty} \frac{(-1)^{n} x^{2n}}{(2n)!},$$
$$\frac{1}{1 - x} = \sum_{n=0}^{\infty} x^{n}\ (|x|<1), \qquad \ln(1 + x) = \sum_{n=1}^{\infty}\frac{(-1)^{n-1} x^{n}}{n}\ (-1 < x \le 1).$$

**Applications.** Taylor expansions let us approximate function values (how a calculator finds $\sin$ or $e^x$), evaluate stubborn limits, and produce series solutions of differential equations. Accuracy is controlled by the **remainder**: the error after truncating at $n$ terms is a quantity we can estimate.

## Worked Examples

**Example 1.** Determine the convergence of $\displaystyle\sum_{n=1}^{\infty}\frac{n}{2^{n}}$.

> **Solution.** A positive series with an exponential $2^n$ — use the ratio test. With $a_n = \dfrac{n}{2^n}$:
> $$\frac{a_{n+1}}{a_n} = \frac{(n+1)/2^{n+1}}{n/2^{n}} = \frac{n+1}{2n} \to \frac{1}{2}.$$
> Since $\rho = \dfrac12 < 1$, the series **converges**.

**Example 2.** Classify $\displaystyle\sum_{n=1}^{\infty}\frac{(-1)^{n-1}}{\sqrt{n}}$ as absolutely convergent, conditionally convergent, or divergent.

> **Solution.** First the absolute series $\sum\dfrac{1}{\sqrt n} = \sum\dfrac{1}{n^{1/2}}$ is a p-series with $p=\frac12\le 1$, so it **diverges** — the original is not absolutely convergent. Now apply the Leibniz test: $u_n=\dfrac1{\sqrt n}$ is decreasing and tends to $0$, so the series converges. Hence it is **conditionally convergent**.

**Example 3.** Find the radius of convergence of $\displaystyle\sum_{n=1}^{\infty}\frac{x^{n}}{n}$.

> **Solution.** The coefficients are $a_n = \dfrac1n$, so by the ratio method:
> $$R = \lim_{n\to\infty}\left|\frac{a_n}{a_{n+1}}\right| = \lim_{n\to\infty}\frac{1/n}{1/(n+1)} = \lim_{n\to\infty}\frac{n+1}{n} = 1.$$
> Thus $R = 1$, with convergence on $(-1,1)$ (the endpoint $x=1$ diverges and $x=-1$ converges, so the interval is $[-1,1)$).

## Exercises

1. Find the sum of the geometric series $\displaystyle\sum_{n=0}^{\infty}\frac{1}{3^{n}}$.
2. Use the p-series rule to decide convergence of $\displaystyle\sum_{n=1}^{\infty}\frac{1}{n^{2}}$ and $\displaystyle\sum_{n=1}^{\infty}\frac{1}{\sqrt{n}}$.
3. Use the ratio test to decide convergence of $\displaystyle\sum_{n=1}^{\infty}\frac{1}{n!}$.
4. Write the first three nonzero terms of the Maclaurin series of $\cos x$.

**Answers.** 1) ratio $r=\frac13$, sum $=\dfrac{1}{1-1/3}=\dfrac32$; 2) the first has $p=2>1$ so it **converges**, the second has $p=\frac12\le1$ so it **diverges**; 3) $\frac{a_{n+1}}{a_n}=\frac{n!}{(n+1)!}=\frac{1}{n+1}\to 0 < 1$, so it **converges** (the sum is in fact $e-1$); 4) $\cos x = 1 - \dfrac{x^2}{2!} + \dfrac{x^4}{4!} - \cdots$, i.e. the first three terms are $1,\ -\dfrac{x^2}{2},\ \dfrac{x^4}{24}$.
