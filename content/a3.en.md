## Core Notes

### Gaussian Elimination

The standard method for solving a linear system is **Gaussian elimination**. Write the system as an **augmented matrix** $[A\mid \vec b]$, use elementary row operations to bring the coefficient part to **row echelon form**, then back-substitute. For example, the system

$$\begin{cases} x + 2y + z = 6, \\ 2x + y - z = 1, \\ x - y + 2z = 5 \end{cases}$$

has augmented matrix $\left[\begin{array}{ccc|c} 1&2&1&6 \\ 2&1&-1&1 \\ 1&-1&2&5 \end{array}\right]$. First clear the first column below the top, then the second column, reaching an upper-triangular echelon form, and **back-substitute**. Reducing further to **reduced row echelon form** (each pivot equal to $1$ with zeros elsewhere in its column) lets you read off the solution directly; this is **Gauss–Jordan elimination**. The first nonzero entry in each row of an echelon form is a **pivot**; its column gives a **pivot variable**, and the other columns give **free variables**. Gaussian elimination costs $O(n^3)$ and is the cornerstone of numerical linear algebra.

**One-step calculation:** for the example above, $R_2-2R_1,\ R_3-R_1$ clear the first column below the top, giving $\left[\begin{array}{ccc|c} 1&2&1&6 \\ 0&-3&-3&-11 \\ 0&-3&1&-1 \end{array}\right]$, ready for further elimination and back-substitution.

### Existence and Uniqueness (via rank)

Let $A\vec x=\vec b$ have $n$ unknowns; write $r=\operatorname{rank}(A)$ for the coefficient rank and $\operatorname{rank}([A\mid\vec b])$ for the augmented rank. Then:

- **No solution (inconsistent)** $\iff \operatorname{rank}(A) < \operatorname{rank}([A\mid\vec b])$, i.e. a contradictory row "$0=$ nonzero" appears;
- **Unique solution** $\iff \operatorname{rank}(A)=\operatorname{rank}([A\mid\vec b])=n$;
- **Infinitely many solutions** $\iff \operatorname{rank}(A)=\operatorname{rank}([A\mid\vec b])=r < n$, with $n-r$ **free variables**.

Memory aid: **first compare the two ranks** (equal $\Rightarrow$ consistent), **then compare with $n$** (equal to $n$ $\Rightarrow$ unique, less than $n$ $\Rightarrow$ infinitely many). A **homogeneous** system $A\vec x=\vec 0$ always has the trivial solution (so it is always consistent); it has a **nonzero solution** iff $r < n$. In particular, a homogeneous system with fewer equations than unknowns ($m<n$) always has a nonzero solution (since $r\le m<n$).

### Linear Dependence and Independence

Given vectors $\vec a_1,\vec a_2,\dots,\vec a_s$, if there exist scalars $k_1,\dots,k_s$ **not all zero** with

$$k_1\vec a_1 + k_2\vec a_2 + \cdots + k_s\vec a_s = \vec 0,$$

the set is **linearly dependent**; otherwise (the equation forces every $k_i=0$) it is **linearly independent**. Test: place the vectors as columns of a matrix — $\operatorname{rank} < s$ means dependent, $\operatorname{rank}=s$ means independent. Intuitively, dependence means some vector is a **linear combination** of the others (it is "redundant", contributing no new direction); independence means every vector is "indispensable". Useful facts: any set containing the **zero vector** is dependent; any $n+1$ vectors in $n$-dimensional space are dependent; a dependent subset forces the whole set dependent, and an independent set has every subset independent.

**One-step calculation:** $\vec a_1=(1,2),\vec a_2=(2,4)$ satisfy $\vec a_2=2\vec a_1$, i.e. $2\vec a_1-\vec a_2=\vec 0$ with nonzero coefficients, hence dependent.

### Basis, Dimension, and Coordinates

A set of vectors in a vector space $V$ that is **linearly independent** and **spans** $V$ (every vector of $V$ is a linear combination of them) is a **basis** of $V$. The number of vectors in a basis is the **dimension** $\dim V$, independent of the chosen basis. The standard basis of $\mathbb{R}^n$ is $\vec e_1,\dots,\vec e_n$, so $\dim\mathbb{R}^n=n$. To check whether $n$ vectors in $\mathbb R^n$ form a basis, just test whether the square matrix with them as columns is **invertible** ($\det\ne0$).

Given a basis $\{\vec b_1,\dots,\vec b_n\}$, every $\vec v\in V$ is written **uniquely** as

$$\vec v = x_1\vec b_1 + x_2\vec b_2 + \cdots + x_n\vec b_n,$$

and the coefficients $(x_1,\dots,x_n)$ are the **coordinates** of $\vec v$ in that basis. Coordinates set up a one-to-one correspondence between abstract vectors and tuples in $\mathbb{R}^n$ — the bridge that turns geometry into computation. Finding coordinates amounts to solving the linear system $B\vec x=\vec v$, where $B=[\vec b_1\ \cdots\ \vec b_n]$.

### Structure of the Solution Set (homogeneous + particular)

The set of all solutions of a homogeneous system $A\vec x=\vec 0$ forms a vector space, the **solution space (null space)**. Its dimension is

$$\dim = n - \operatorname{rank}(A),$$

which is the **rank–nullity theorem**. A basis of the solution space is a **fundamental system of solutions** $\vec\xi_1,\dots,\vec\xi_{n-r}$, and the homogeneous general solution is

$$\vec x = c_1\vec\xi_1 + \cdots + c_{n-r}\vec\xi_{n-r}.$$

For an **inhomogeneous** system $A\vec x=\vec b$, the general solution = **one particular solution** $\vec x_0$ + **the general solution of the associated homogeneous system**:

$$\vec x = \vec x_0 + c_1\vec\xi_1 + \cdots + c_{n-r}\vec\xi_{n-r}.$$

Geometrically the inhomogeneous solution set is an **affine subspace** through $\vec x_0$ parallel to the null space (a translated "plane"). This structure theorem splits "find all solutions" into "find one particular solution" + "find a fundamental system", which is highly practical.

### Summary

- Gaussian elimination: augmented matrix $\to$ row echelon form $\to$ back-substitute; reduced row echelon form is read off directly ($O(n^3)$).
- Solvability: $\operatorname{rank}(A)<\operatorname{rank}([A\mid\vec b])$ $\Rightarrow$ no solution; equal ranks $=n$ $\Rightarrow$ unique; equal ranks $=r<n$ $\Rightarrow$ $n-r$ free variables, infinitely many.
- Homogeneous systems are always consistent; nonzero solution $\iff r<n$; $m<n$ forces a nonzero solution.
- Linear dependence $\iff\operatorname{rank}<s\iff$ some vector is a combination of the rest; any set with the zero vector is dependent; $n+1$ vectors in $\mathbb R^n$ are dependent.
- Basis = independent + spanning; $\dim$ is basis-independent; coordinates are the unique solution of $B\vec x=\vec v$.
- Rank–nullity: $\dim(\text{null space})=n-\operatorname{rank}(A)$; inhomogeneous general solution = particular + homogeneous general.

## Worked Examples

**Example 1.** Solve by Gaussian elimination:
$$\begin{cases} x + y + z = 6, \\ 2x - y + z = 3, \\ x + 2y - z = 2. \end{cases}$$

> **Solution.** Row-reduce the augmented matrix:
> $$\left[\begin{array}{ccc|c} 1&1&1&6 \\ 2&-1&1&3 \\ 1&2&-1&2 \end{array}\right] \xrightarrow{R_2-2R_1,\ R_3-R_1} \left[\begin{array}{ccc|c} 1&1&1&6 \\ 0&-3&-1&-9 \\ 0&1&-2&-4 \end{array}\right].$$
> Swap $R_2,R_3$ and eliminate with $R_3+3R_2$:
> $$\xrightarrow{\ \ } \left[\begin{array}{ccc|c} 1&1&1&6 \\ 0&1&-2&-4 \\ 0&0&-7&-21 \end{array}\right].$$
> Back-substitute: the last row gives $-7z=-21$, so $z=3$; then $y-2z=-4$ gives $y=2$; finally $x+y+z=6$ gives $x=1$. The solution is $(x,y,z)=(1,2,3)$.

**Example 2.** Determine whether $\vec a_1=(1,2,3)$, $\vec a_2=(2,4,6)$, $\vec a_3=(1,0,1)$ are linearly dependent.

> **Solution.** Form a matrix with the vectors as columns and find its rank:
> $$\begin{bmatrix} 1 & 2 & 1 \\ 2 & 4 & 0 \\ 3 & 6 & 1 \end{bmatrix} \xrightarrow{R_2-2R_1,\ R_3-3R_1} \begin{bmatrix} 1 & 2 & 1 \\ 0 & 0 & -2 \\ 0 & 0 & -2 \end{bmatrix} \xrightarrow{R_3-R_2} \begin{bmatrix} 1 & 2 & 1 \\ 0 & 0 & -2 \\ 0 & 0 & 0 \end{bmatrix}.$$
> The rank is $2 < 3$, so the set is **linearly dependent**. Indeed $\vec a_2 = 2\vec a_1$, which is plainly a dependence.

**Example 3.** Find a fundamental system of solutions and the general solution of $\begin{cases} x_1 + x_2 - x_3 + x_4 = 0 \\ 2x_1 + 2x_2 - x_3 + 3x_4 = 0 \end{cases}$.

> **Solution.** Reduce the coefficient matrix (constants are zero):
> $$\begin{bmatrix} 1 & 1 & -1 & 1 \\ 2 & 2 & -1 & 3 \end{bmatrix} \xrightarrow{R_2-2R_1} \begin{bmatrix} 1 & 1 & -1 & 1 \\ 0 & 0 & 1 & 1 \end{bmatrix} \xrightarrow{R_1+R_2} \begin{bmatrix} 1 & 1 & 0 & 2 \\ 0 & 0 & 1 & 1 \end{bmatrix}.$$
> Pivot variables are $x_1,x_3$; free variables are $x_2,x_4$ (since $n-r=4-2=2$). From the reduced form, $x_1=-x_2-2x_4$ and $x_3=-x_4$.
> Setting $(x_2,x_4)=(1,0)$ gives $\vec\xi_1=(-1,1,0,0)$; setting $(x_2,x_4)=(0,1)$ gives $\vec\xi_2=(-2,0,-1,1)$.
> The general solution is $\vec x = c_1\vec\xi_1 + c_2\vec\xi_2 = c_1(-1,1,0,0) + c_2(-2,0,-1,1)$ for arbitrary $c_1,c_2$.

**Example 4.** Find the general solution (particular + homogeneous) of $\begin{cases} x_1+x_2+x_3+x_4=4 \\ x_1+2x_2+3x_3+4x_4=10 \end{cases}$.

> **Solution.** Reduce the augmented matrix to reduced echelon form:
> $$\left[\begin{array}{cccc|c} 1&1&1&1&4 \\ 1&2&3&4&10 \end{array}\right] \xrightarrow{R_2-R_1} \left[\begin{array}{cccc|c} 1&1&1&1&4 \\ 0&1&2&3&6 \end{array}\right] \xrightarrow{R_1-R_2} \left[\begin{array}{cccc|c} 1&0&-1&-2&-2 \\ 0&1&2&3&6 \end{array}\right].$$
> Here $\operatorname{rank}(A)=\operatorname{rank}([A\mid\vec b])=2<4$, so there are infinitely many solutions with free variables $x_3,x_4$.
> From the reduced form: $x_1=-2+x_3+2x_4$ and $x_2=6-2x_3-3x_4$.
> Setting $x_3=x_4=0$ gives the **particular solution** $\vec x_0=(-2,6,0,0)$. For the homogeneous part, $x_3=1,x_4=0$ gives $\vec\xi_1=(1,-2,1,0)$ and $x_3=0,x_4=1$ gives $\vec\xi_2=(2,-3,0,1)$.
> Hence $\vec x=(-2,6,0,0)+c_1(1,-2,1,0)+c_2(2,-3,0,1)$. Substituting back, $\vec x_0$ gives the right side $(4,10)$ and each $\vec\xi_i$ gives $(0,0)$, confirming the result.

**Example 5.** Given that $\vec b_1=(1,1,1)$, $\vec b_2=(1,1,0)$, $\vec b_3=(1,0,0)$ form a basis of $\mathbb R^3$, find the coordinates of $\vec v=(3,2,1)$ in this basis.

> **Solution.** Write $\vec v=x_1\vec b_1+x_2\vec b_2+x_3\vec b_3$, i.e. solve $\begin{bmatrix}1&1&1\\1&1&0\\1&0&0\end{bmatrix}\begin{bmatrix}x_1\\x_2\\x_3\end{bmatrix}=\begin{bmatrix}3\\2\\1\end{bmatrix}$.
> The third row gives $x_1=1$; the second row $x_1+x_2=2$ gives $x_2=1$; the first row $x_1+x_2+x_3=3$ gives $x_3=1$.
> So the coordinates are $(1,1,1)$, i.e. $\vec v=\vec b_1+\vec b_2+\vec b_3$. (Since $\det[\vec b_1\,\vec b_2\,\vec b_3]=-1\ne0$, these three vectors do form a basis and the coordinates are unique.)

## Exercises

1. Solve $\begin{cases} x + y = 3 \\ x - y + z = 2 \\ 2x + y + z = 7 \end{cases}$ by Gaussian elimination.
2. Decide whether $\begin{cases} x + 2y = 1 \\ 2x + 4y = 3 \end{cases}$ is solvable, and justify with rank.
3. Decide whether $(1,0,1),(0,1,1),(1,1,2)$ are linearly independent; can they form a basis of $\mathbb{R}^3$?
4. Find a fundamental system of solutions of $\begin{cases} x_1 - x_2 + 2x_3 = 0 \\ 2x_1 - 2x_2 + 4x_3 = 0 \end{cases}$.
5. With parameter $a$, classify the solutions (none / unique / infinitely many) of $\begin{cases} x+y-z=1 \\ x-y+2z=2 \\ 2x+az=3 \end{cases}$.
6. Determine whether $\vec a_1=(1,2,3),\vec a_2=(2,3,4),\vec a_3=(3,5,7),\vec a_4=(1,1,1)$ are linearly dependent; if so, give one nontrivial linear relation.

**Answers.**
1) From $x+y=3$, $x=3-y$; substituting into the other two equations gives $(x,y,z)=(1,2,3)$.
2) $\operatorname{rank}(A)=1$ but $\operatorname{rank}([A\mid b])=2$; since they differ the system is **inconsistent** (the second equation doubles the first but the right-hand sides are not in proportion).
3) With the three vectors as columns, $\begin{vmatrix}1&0&1\\0&1&1\\1&1&2\end{vmatrix}=0$, so the rank is $2<3$: they are **linearly dependent** and cannot form a basis (indeed $(1,1,2)=(1,0,1)+(0,1,1)$).
4) The rank is $1$ with free variables $x_2,x_3$; from $x_1=x_2-2x_3$ we get $\vec\xi_1=(1,1,0)$ and $\vec\xi_2=(-2,0,1)$.
5) The coefficient determinant is $\det A=\begin{vmatrix}1&1&-1\\1&-1&2\\2&0&a\end{vmatrix}=-2(a-1)$. When $a\ne1$, $\det A\ne0$, giving a **unique solution** (e.g. for $a=2$ it is $(\tfrac32,-\tfrac12,0)$); when $a=1$, $\det A=0$ and the third equation equals the sum of the first two, so $\operatorname{rank}(A)=\operatorname{rank}([A\mid b])=2<3$, giving **infinitely many solutions**. The system is never inconsistent here.
6) Four vectors in $\mathbb R^3$ are necessarily dependent. Reducing with them as columns gives $\operatorname{rank}=2$ and the relation $\vec a_1+\vec a_2-\vec a_3=\vec 0$ (i.e. $(1,2,3)+(2,3,4)=(3,5,7)$), a nontrivial linear relation.
