## 核心讲义

极限是整个微积分的逻辑起点。导数是一种极限，定积分是另一种极限，无穷级数的和还是极限。把「无限接近」这件事说清楚，后面的一切才站得住脚。

### 数列极限与收敛

一个**数列** $\{a_n\}$ 就是按下标排好的一串数 $a_1, a_2, a_3, \dots$。我们关心当 $n$ 越来越大时，$a_n$ 是否会稳定地靠近某个固定的数。

**定义（数列极限）.** 若存在常数 $A$，使得对任意给定的 $\varepsilon > 0$，都能找到正整数 $N$，当 $n > N$ 时恒有
$$|a_n - A| < \varepsilon,$$
则称数列 $\{a_n\}$ **收敛**于 $A$，记作 $\lim_{n\to\infty} a_n = A$。否则称 $\{a_n\}$ **发散**。

直观地说：无论你把误差带 $(A-\varepsilon,\ A+\varepsilon)$ 取得多窄，从某一项 $N$ 之后，数列的所有项都掉进这条带子里。$\varepsilon$ 是「挑战」，$N$ 是「应答」——能对任意挑战给出应答，就是收敛。

收敛数列的两个基本性质：**极限唯一**；**收敛必有界**（反之不真，如 $a_n=(-1)^n$ 有界却发散）。

### 函数极限的 ε–δ 定义

数列只在整数点取值，函数则在连续区间上变化，所以要把「$n\to\infty$」换成「$x\to x_0$」。

**定义（函数极限）.** 设 $f$ 在 $x_0$ 的某去心邻域内有定义。若存在常数 $A$，使得对任意 $\varepsilon>0$，都存在 $\delta>0$，当 $0 < |x - x_0| < \delta$ 时恒有
$$|f(x) - A| < \varepsilon,$$
则称 $x\to x_0$ 时 $f$ 以 $A$ 为极限，记 $\lim_{x\to x_0} f(x) = A$。

注意 $0 < |x-x_0|$ 这个条件：极限只关心 $x_0$ **附近**的行为，与 $f$ 在 $x_0$ 处取什么值、甚至有没有定义，毫无关系。这正是极限和「直接代入」的本质区别。

**单侧极限**：只让 $x$ 从右边趋近写作 $\lim_{x\to x_0^+}f(x)$，从左边趋近写作 $\lim_{x\to x_0^-}f(x)$。双侧极限存在当且仅当两个单侧极限存在且相等。

### 极限运算法则

若 $\lim f = A$、$\lim g = B$（同一个趋近过程），则

$$\lim (f \pm g) = A \pm B, \qquad \lim (f\cdot g) = A\cdot B, \qquad \lim \frac{f}{g} = \frac{A}{B}\ (B \ne 0).$$

这些法则把复杂表达式拆成简单部件分别求极限。但要警惕**未定式**：当出现 $\frac{0}{0}$、$\frac{\infty}{\infty}$、$\infty-\infty$、$0\cdot\infty$ 这类形式时，不能直接套用，必须先变形（约分、通分、有理化、提取主部）。

**夹逼定理（两边夹）.** 若在 $x_0$ 附近有 $g(x) \le f(x) \le h(x)$，且 $\lim g(x) = \lim h(x) = A$，则 $\lim f(x) = A$。它特别适合处理被三角函数或符号震荡「卡住」的极限。

### 两个重要极限

这两个极限是后续大量公式的源头，必须记牢。

**第一个重要极限：**
$$\lim_{x\to 0} \frac{\sin x}{x} = 1.$$
它说明当 $x$ 很小时 $\sin x \approx x$，是所有三角函数求导的基石。证明用到单位圆上的面积夹逼：$\sin x < x < \tan x$。

**第二个重要极限：**
$$\lim_{x\to\infty}\left(1 + \frac{1}{x}\right)^{x} = e.$$
等价形式（令 $t=\frac1x$）为 $\lim_{t\to 0}(1+t)^{1/t} = e$。它是自然常数 $e\approx 2.71828$ 的定义来源，处理「$1^{\infty}$」型未定式时几乎都要回到这个形式。

### 无穷小与等价替换

**定义.** 若 $\lim_{x\to x_0}\alpha(x) = 0$，则称 $\alpha$ 为该过程下的**无穷小**。无穷小不是「很小的数」，而是一个趋于零的变量。

两个无穷小相比，可以快慢不同。设 $\alpha,\beta$ 都是无穷小：若 $\lim\frac{\alpha}{\beta}=1$，称 $\alpha$ 与 $\beta$ **等价**，记 $\alpha\sim\beta$。等价无穷小可以在**乘除因子**中互相替换，从而极大简化计算（但加减法中替换需谨慎）。

当 $x\to 0$ 时，常用的等价无穷小有：
$$\sin x \sim x,\quad \tan x \sim x,\quad 1-\cos x \sim \frac{x^2}{2},\quad \ln(1+x)\sim x,\quad e^x - 1 \sim x,\quad (1+x)^a - 1 \sim a x.$$

### 连续性与间断点

**定义（连续）.** 若 $\lim_{x\to x_0} f(x) = f(x_0)$，则称 $f$ 在 $x_0$ **连续**。这一个等式同时要求三件事：$f(x_0)$ 有定义、极限存在、二者相等。

直观上，连续就是「函数图像在该点不断开、可以一笔画过」。在闭区间上连续的函数有两条强有力的性质：**最值定理**（必取到最大、最小值）与**介值定理**（取遍两端点函数值之间的一切值）。

若 $f$ 在 $x_0$ 不连续，则 $x_0$ 是**间断点**，分两类：

- **第一类间断点**：左右极限都存在。若两者相等（但不等于 $f(x_0)$ 或 $f$ 在此无定义）称**可去间断点**；若两者不等称**跳跃间断点**。
- **第二类间断点**：左右极限至少有一个不存在（如趋于 $\infty$ 的**无穷间断点**，或像 $\sin\frac1x$ 那样的**振荡间断点**）。

## 典型例题

**例 1.** 求 $\displaystyle \lim_{x\to 3}\frac{x^2 - 9}{x - 3}$。

> **解.** 直接代入得 $\frac{0}{0}$，是未定式，需先约分。注意 $x\to 3$ 时 $x\ne 3$，故 $x-3\ne 0$ 可约去：
> $$\frac{x^2-9}{x-3} = \frac{(x-3)(x+3)}{x-3} = x + 3.$$
> 于是
> $$\lim_{x\to 3}\frac{x^2-9}{x-3} = \lim_{x\to 3}(x+3) = 6.$$

**例 2.** 求 $\displaystyle \lim_{x\to 0}\frac{\sin 5x}{\tan 3x}$。

> **解.** 用 $x\to0$ 时的等价无穷小 $\sin 5x \sim 5x$、$\tan 3x \sim 3x$（它们都是乘除因子，可替换）：
> $$\lim_{x\to 0}\frac{\sin 5x}{\tan 3x} = \lim_{x\to 0}\frac{5x}{3x} = \frac{5}{3}.$$
> 若不用等价替换，也可凑第一个重要极限：
> $$\frac{\sin 5x}{\tan 3x} = \frac{\sin 5x}{5x}\cdot\frac{3x}{\tan 3x}\cdot\frac{5}{3} \to 1\cdot 1 \cdot \frac{5}{3} = \frac{5}{3}.$$

**例 3.** 求 $\displaystyle \lim_{x\to\infty}\left(1 + \frac{2}{x}\right)^{x}$。

> **解.** 这是 $1^{\infty}$ 型，要凑成第二个重要极限的样子。令底数里的分子配成 $\frac1u$：取 $u = \frac{x}{2}$，则 $x = 2u$，且 $x\to\infty$ 时 $u\to\infty$：
> $$\left(1+\frac{2}{x}\right)^{x} = \left(1+\frac{1}{u}\right)^{2u} = \left[\left(1+\frac{1}{u}\right)^{u}\right]^{2}.$$
> 内层 $\to e$，故整体
> $$\lim_{x\to\infty}\left(1+\frac{2}{x}\right)^{x} = e^{2}.$$

## 练习与自测

1. 用 $\varepsilon$–$N$ 思想说明 $\displaystyle\lim_{n\to\infty}\frac{1}{n}=0$：对 $\varepsilon>0$，应取 $N$ 为多大？
2. 求 $\displaystyle \lim_{x\to 2}\frac{x^2 - 4}{x^2 - 3x + 2}$。
3. 求 $\displaystyle \lim_{x\to 0}\frac{1-\cos x}{x^2}$。
4. 讨论 $f(x)=\dfrac{|x|}{x}$ 在 $x=0$ 处的连续性，并指出间断点的类型。

**参考答案.** 1) 由 $\left|\frac1n-0\right|=\frac1n<\varepsilon$ 解得 $n>\frac1\varepsilon$，故取 $N=\left[\frac1\varepsilon\right]$（取整）即可；2) 分子分母同时分解，$\frac{(x-2)(x+2)}{(x-1)(x-2)}\to\frac{x+2}{x-1}\Big|_{x=2}=4$；3) 用 $1-\cos x\sim\frac{x^2}{2}$，原式 $=\frac12$；4) 左极限 $-1$、右极限 $+1$，两侧极限存在但不等，故 $x=0$ 是**跳跃间断点**（第一类），$f$ 在该点不连续。
