## 核心讲义

极限是整个微积分的逻辑起点。导数是一种极限，定积分是另一种极限，无穷级数的和还是极限。把「无限接近」这件事说清楚，后面的一切才站得住脚。本章的核心任务，就是把直觉上的「越来越近」翻译成无可争辩的不等式语言。

### 数列极限与收敛

一个**数列** $\{a_n\}$ 就是按下标排好的一串数 $a_1, a_2, a_3, \dots$。我们关心当 $n$ 越来越大时，$a_n$ 是否会稳定地靠近某个固定的数。

**定义（数列极限）.** 若存在常数 $A$，使得对任意给定的 $\varepsilon > 0$，都能找到正整数 $N$，当 $n > N$ 时恒有
$$|a_n - A| < \varepsilon,$$
则称数列 $\{a_n\}$ **收敛**于 $A$，记作 $\lim_{n\to\infty} a_n = A$。否则称 $\{a_n\}$ **发散**。

直观地说：无论你把误差带 $(A-\varepsilon,\ A+\varepsilon)$ 取得多窄，从某一项 $N$ 之后，数列的所有项都掉进这条带子里。$\varepsilon$ 是「挑战」，$N$ 是「应答」——能对任意挑战给出应答，就是收敛。关键在于 $N$ **允许依赖** $\varepsilon$：$\varepsilon$ 越小，往往要走得越远（$N$ 越大），但只要总能找到，就算赢。

收敛数列的两个基本性质：**极限唯一**（若同时收敛到 $A$ 和 $B$，取 $\varepsilon=\frac{|A-B|}{2}$ 会导出矛盾）；**收敛必有界**（反之不真，如 $a_n=(-1)^n$ 有界却发散）。一个常用的充分条件是**单调有界准则**：单调递增且有上界的数列必收敛——第二个重要极限正是靠它保证 $e$ 存在。

### 函数极限的 ε–δ 定义

数列只在整数点取值，函数则在连续区间上变化，所以要把「$n\to\infty$」换成「$x\to x_0$」。

**定义（函数极限）.** 设 $f$ 在 $x_0$ 的某去心邻域内有定义。若存在常数 $A$，使得对任意 $\varepsilon>0$，都存在 $\delta>0$，当 $0 < |x - x_0| < \delta$ 时恒有
$$|f(x) - A| < \varepsilon,$$
则称 $x\to x_0$ 时 $f$ 以 $A$ 为极限，记 $\lim_{x\to x_0} f(x) = A$。

注意 $0 < |x-x_0|$ 这个条件：极限只关心 $x_0$ **附近**的行为，与 $f$ 在 $x_0$ 处取什么值、甚至有没有定义，毫无关系。这正是极限和「直接代入」的本质区别。$\delta$ 扮演数列里 $N$ 的角色：它度量「$x$ 要离 $x_0$ 多近」，同样允许依赖 $\varepsilon$。

**单侧极限**：只让 $x$ 从右边趋近写作 $\lim_{x\to x_0^+}f(x)$，从左边趋近写作 $\lim_{x\to x_0^-}f(x)$。双侧极限存在当且仅当两个单侧极限存在且相等。这条判据是判定分段函数衔接处、以及含 $|x|$ 函数极限的标准工具。

### 极限运算法则

若 $\lim f = A$、$\lim g = B$（同一个趋近过程），则

$$\lim (f \pm g) = A \pm B, \qquad \lim (f\cdot g) = A\cdot B, \qquad \lim \frac{f}{g} = \frac{A}{B}\ (B \ne 0).$$

这些法则把复杂表达式拆成简单部件分别求极限。但要警惕**未定式**：当出现 $\frac{0}{0}$、$\frac{\infty}{\infty}$、$\infty-\infty$、$0\cdot\infty$ 这类形式时，不能直接套用，必须先变形（约分、通分、有理化、提取主部）。处理 $x\to\infty$ 的有理分式时，常用技巧是分子分母同除以分母的最高次幂，把不重要的项压成无穷小。

**夹逼定理（两边夹）.** 若在 $x_0$ 附近有 $g(x) \le f(x) \le h(x)$，且 $\lim g(x) = \lim h(x) = A$，则 $\lim f(x) = A$。它特别适合处理被三角函数或符号震荡「卡住」的极限，比如 $x\sin\frac1x$：用 $-|x|\le x\sin\frac1x\le|x|$ 一夹即得极限为 $0$。

### 两个重要极限

这两个极限是后续大量公式的源头，必须记牢。

**第一个重要极限：**
$$\lim_{x\to 0} \frac{\sin x}{x} = 1.$$
它说明当 $x$ 很小时 $\sin x \approx x$，是所有三角函数求导的基石。证明用到单位圆上的面积夹逼：在第一象限有 $\sin x < x < \tan x$，三式同除 $\sin x$ 得 $1 < \dfrac{x}{\sin x} < \dfrac{1}{\cos x}$，再令 $x\to 0$ 夹逼即可。

**第二个重要极限：**
$$\lim_{x\to\infty}\left(1 + \frac{1}{x}\right)^{x} = e.$$
等价形式（令 $t=\frac1x$）为 $\lim_{t\to 0}(1+t)^{1/t} = e$。它是自然常数 $e\approx 2.71828$ 的定义来源，处理「$1^{\infty}$」型未定式时几乎都要回到这个形式。通法是把底凑成 $1+\frac1u$、把指数凑成 $u$ 的常数倍，或取对数转为 $\frac{0}{0}$ 处理。

### 无穷小与等价替换

**定义.** 若 $\lim_{x\to x_0}\alpha(x) = 0$，则称 $\alpha$ 为该过程下的**无穷小**。无穷小不是「很小的数」，而是一个趋于零的变量。与之对偶，绝对值无限增大的变量称为**无穷大**；二者关系是：非零无穷小的倒数是无穷大，反之亦然。

两个无穷小相比，可以快慢不同。设 $\alpha,\beta$ 都是无穷小：若 $\lim\frac{\alpha}{\beta}=1$，称 $\alpha$ 与 $\beta$ **等价**，记 $\alpha\sim\beta$；若 $\lim\frac{\alpha}{\beta}=0$，称 $\alpha$ 是比 $\beta$ **高阶**的无穷小，记 $\alpha=o(\beta)$。等价无穷小可以在**乘除因子**中互相替换，从而极大简化计算（但加减法中替换需谨慎，否则会丢掉主部、得出错误结果）。

当 $x\to 0$ 时，常用的等价无穷小有：
$$\sin x \sim x,\quad \tan x \sim x,\quad \arcsin x \sim x,\quad 1-\cos x \sim \frac{x^2}{2},\quad \ln(1+x)\sim x,\quad e^x - 1 \sim x,\quad (1+x)^a - 1 \sim a x.$$

### 连续性与间断点

**定义（连续）.** 若 $\lim_{x\to x_0} f(x) = f(x_0)$，则称 $f$ 在 $x_0$ **连续**。这一个等式同时要求三件事：$f(x_0)$ 有定义、极限存在、二者相等。等价地，记增量 $\Delta y=f(x_0+\Delta x)-f(x_0)$，连续即 $\lim_{\Delta x\to 0}\Delta y=0$。

直观上，连续就是「函数图像在该点不断开、可以一笔画过」。初等函数（多项式、有理式、指数、对数、三角及其复合）在各自定义域内处处连续，这给「直接代入求极限」提供了合法依据。在闭区间上连续的函数有两条强有力的性质：**最值定理**（必取到最大、最小值）与**介值定理**（取遍两端点函数值之间的一切值）。介值定理是方程求根「存在性」的理论根据：若 $f$ 连续且 $f(a)f(b)<0$，则 $(a,b)$ 内必有零点。

若 $f$ 在 $x_0$ 不连续，则 $x_0$ 是**间断点**，分两类：

- **第一类间断点**：左右极限都存在。若两者相等（但不等于 $f(x_0)$ 或 $f$ 在此无定义）称**可去间断点**；若两者不等称**跳跃间断点**。
- **第二类间断点**：左右极限至少有一个不存在（如趋于 $\infty$ 的**无穷间断点**，或像 $\sin\frac1x$ 那样的**振荡间断点**）。

### 小结

- 极限的精确语言：数列用 $\varepsilon$–$N$，函数用 $\varepsilon$–$\delta$；核心是「任意 $\varepsilon$，存在对应的 $N$ 或 $\delta$」。
- 收敛数列**唯一且有界**；**单调有界必收敛**。双侧极限存在 $\Leftrightarrow$ 左右极限存在且相等。
- 四则运算法则只在极限都存在（除法分母非零）时成立；遇未定式先变形。
- 两个重要极限：$\displaystyle\lim_{x\to0}\frac{\sin x}{x}=1$，$\displaystyle\lim_{x\to\infty}\left(1+\frac1x\right)^x=e$。
- 常用等价无穷小（$x\to0$）：$\sin x\sim x$，$\tan x\sim x$，$1-\cos x\sim\frac{x^2}{2}$，$\ln(1+x)\sim x$，$e^x-1\sim x$，$(1+x)^a-1\sim ax$；只在乘除因子中替换。
- 连续 $\Leftrightarrow\lim_{x\to x_0}f(x)=f(x_0)$；间断点按左右极限是否存在分第一类（可去/跳跃）与第二类（无穷/振荡）。

## 典型例题

**例 1.** 求 $\displaystyle \lim_{x\to 3}\frac{x^2 - 9}{x - 3}$。

> **解.** 直接代入得 $\frac{0}{0}$，是未定式，需先约分。注意 $x\to 3$ 时 $x\ne 3$，故 $x-3\ne 0$ 可约去：
> $$\frac{x^2-9}{x-3} = \frac{(x-3)(x+3)}{x-3} = x + 3.$$
> 于是
> $$\lim_{x\to 3}\frac{x^2-9}{x-3} = \lim_{x\to 3}(x+3) = 6.$$

**例 2.** 求 $\displaystyle \lim_{x\to 0}\frac{\sin 5x}{\tan 3x}$。

> **解.** 用 $x\to0$ 时的等价无穷小 $\sin 5x \sim 5x$、$\tan 3x \sim 3x$（它们都是乘除因子，可替换）：
> $$\lim_{x\to 0}\frac{\sin 5x}{\tan 3x} = \lim_{x\to 0}\frac{5x}{3x} = \frac{5}{3}.$$
> 若不用等价替换，也可凑第一个重要极限：
> $$\frac{\sin 5x}{\tan 3x} = \frac{\sin 5x}{5x}\cdot\frac{3x}{\tan 3x}\cdot\frac{5}{3} \to 1\cdot 1 \cdot \frac{5}{3} = \frac{5}{3}.$$

**例 3.** 求 $\displaystyle \lim_{x\to\infty}\left(1 + \frac{2}{x}\right)^{x}$。

> **解.** 这是 $1^{\infty}$ 型，要凑成第二个重要极限的样子。令底数里的分子配成 $\frac1u$：取 $u = \frac{x}{2}$，则 $x = 2u$，且 $x\to\infty$ 时 $u\to\infty$：
> $$\left(1+\frac{2}{x}\right)^{x} = \left(1+\frac{1}{u}\right)^{2u} = \left[\left(1+\frac{1}{u}\right)^{u}\right]^{2}.$$
> 内层 $\to e$，故整体
> $$\lim_{x\to\infty}\left(1+\frac{2}{x}\right)^{x} = e^{2}.$$

**例 4.** 求 $\displaystyle \lim_{x\to +\infty}\left(\sqrt{x^2 + x} - x\right)$。

> **解.** 这是 $\infty - \infty$ 型，需先变形消除相消。用共轭有理化，分子分母同乘 $\sqrt{x^2+x}+x$：
> $$\sqrt{x^2+x}-x = \frac{(x^2+x) - x^2}{\sqrt{x^2+x}+x} = \frac{x}{\sqrt{x^2+x}+x}.$$
> 分子分母同除以 $x$（注意 $x>0$ 时 $\sqrt{x^2}=x$）：
> $$= \frac{1}{\sqrt{1+\frac1x}+1}\ \longrightarrow\ \frac{1}{\sqrt{1+0}+1} = \frac{1}{2}.$$
> 所以极限为 $\dfrac12$。

**例 5.** 设 $f(x)=\begin{cases} \dfrac{\sin x}{x}, & x < 0,\\[4pt] a, & x = 0,\\[4pt] x + b, & x > 0,\end{cases}$ 问 $a,b$ 取何值时 $f$ 在 $x=0$ 处连续？

> **解.** 连续要求左极限、右极限、函数值三者相等。
> 左极限：$\displaystyle\lim_{x\to 0^-}\frac{\sin x}{x} = 1$（第一个重要极限）。
> 右极限：$\displaystyle\lim_{x\to 0^+}(x+b) = b$。
> 函数值：$f(0) = a$。
> 要在 $x=0$ 连续，须三者相等：$1 = b = a$。故当 $a = 1$ 且 $b = 1$ 时 $f$ 在 $x=0$ 连续。

## 练习与自测

1. 用 $\varepsilon$–$N$ 思想说明 $\displaystyle\lim_{n\to\infty}\frac{1}{n}=0$：对 $\varepsilon>0$，应取 $N$ 为多大？
2. 求 $\displaystyle \lim_{x\to 2}\frac{x^2 - 4}{x^2 - 3x + 2}$。
3. 求 $\displaystyle \lim_{x\to 0}\frac{1-\cos x}{x^2}$。
4. 求 $\displaystyle \lim_{x\to\infty}\frac{3x^2 - x + 1}{2x^2 + 5}$。
5. 求 $\displaystyle \lim_{x\to 0}\frac{\ln(1+2x)}{e^{3x}-1}$。
6. 讨论 $f(x)=\dfrac{|x|}{x}$ 在 $x=0$ 处的连续性，并指出间断点的类型。

**参考答案.** 1) 由 $\left|\frac1n-0\right|=\frac1n<\varepsilon$ 解得 $n>\frac1\varepsilon$，故取 $N=\left[\frac1\varepsilon\right]$（取整）即可；2) 分子分母同时分解，$\frac{(x-2)(x+2)}{(x-1)(x-2)}\to\frac{x+2}{x-1}\Big|_{x=2}=4$；3) 用 $1-\cos x\sim\frac{x^2}{2}$，原式 $=\frac12$；4) 分子分母同除 $x^2$，得 $\frac{3-1/x+1/x^2}{2+5/x^2}\to\frac32$；5) 用等价无穷小 $\ln(1+2x)\sim 2x$、$e^{3x}-1\sim 3x$，原式 $=\lim\frac{2x}{3x}=\frac23$；6) 左极限 $-1$、右极限 $+1$，两侧极限存在但不等，故 $x=0$ 是**跳跃间断点**（第一类），$f$ 在该点不连续。
