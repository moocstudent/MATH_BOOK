## 核心讲义

### 假设检验的基本思想

**假设检验**用样本对关于总体的某个命题作出"拒绝"或"不拒绝"的判断。

待判断的命题写成一对对立假设:**原假设** $H_0$ 与**备择假设** $H_1$。通常把"维持现状/无效应/想保护的稳妥结论"设为 $H_0$,把"有差异/有效应"设为 $H_1$。例如检验某工艺均值是否仍为 $\mu_0$,即在 $H_0:\mu=\mu_0$ 与 $H_1:\mu\ne\mu_0$ 之间作判断。

**基本逻辑(反证 + 小概率原理).** 先假定 $H_0$ 成立,构造一个分布已知的**检验统计量**;若由样本算出的值落入"在 $H_0$ 下几乎不可能出现"的范围(**拒绝域**),就认为出现了小概率事件,从而**拒绝 $H_0$**;否则**不拒绝 $H_0$**。这里的"几乎不可能"用**显著性水平** $\alpha$(如 $0.05,0.01$)量化:拒绝域的概率(在 $H_0$ 下)恰为 $\alpha$。

**p 值.** 在 $H_0$ 成立的前提下,出现"比观测结果更极端"数据的概率,称为 **p 值**。判别法则:$p\le\alpha$ 则拒绝 $H_0$,$p>\alpha$ 则不拒绝。p 值越小,反对 $H_0$ 的证据越强。

**单双侧.** $H_1:\mu\ne\mu_0$ 为**双侧**,拒绝域在两端;$H_1:\mu>\mu_0$(或 $<$)为**单侧**,拒绝域在一端。

### 两类错误与检验功效

判断可能出错,有两种:

**第一类错误(弃真).** $H_0$ 实际为真却被拒绝。其概率(在 $H_0$ 为真的条件下拒绝 $H_0$)记为 $\alpha=P(\text{reject } H_0\mid H_0)$,正是显著性水平。

**第二类错误(取伪).** $H_0$ 实际为假却未被拒绝。其概率(在 $H_1$ 为真的条件下不拒绝 $H_0$)记为 $\beta=P(\text{accept } H_0\mid H_1)$。

| | $H_0$ 为真 | $H_0$ 为假 |
|---|---|---|
| 拒绝 $H_0$ | 第一类错误($\alpha$) | 判断正确 |
| 不拒绝 $H_0$ | 判断正确 | 第二类错误($\beta$) |

**功效(power).** $1-\beta$ 称为检验的**功效**,即当 $H_1$ 为真时正确拒绝 $H_0$ 的概率,越大越好。

**权衡.** 样本量固定时,减小 $\alpha$ 会增大 $\beta$,二者此消彼长,不能同时任意减小。实务上先固定 $\alpha$(控制弃真),再通过增大样本量 $n$ 来减小 $\beta$、提高功效。这解释了为何"不拒绝 $H_0$"不等于"$H_0$ 成立"——可能只是样本不足以察觉差异。

### 均值的检验(z/t 检验)

设总体 $X\sim N(\mu,\sigma^2)$,检验 $\mu$ 是否等于已知常数 $\mu_0$,水平 $\alpha$。

**(1) $\sigma^2$ 已知:$z$ 检验(u 检验).** 统计量
$$Z=\frac{\bar X-\mu_0}{\sigma/\sqrt n}\overset{H_0}{\sim} N(0,1).$$
拒绝域:双侧 $|Z|>z_{\alpha/2}$;右侧($H_1:\mu>\mu_0$)$Z>z_\alpha$;左侧 $Z<-z_\alpha$。

**(2) $\sigma^2$ 未知:$t$ 检验.** 用 $S$ 代 $\sigma$,
$$T=\frac{\bar X-\mu_0}{S/\sqrt n}\overset{H_0}{\sim} t(n-1).$$
拒绝域:双侧 $|T|>t_{\alpha/2}(n-1)$;右侧 $T>t_\alpha(n-1)$;左侧 $T<-t_\alpha(n-1)$。

**配对/两样本.** 比较两正态总体均值差 $\mu_1-\mu_2$:方差已知用两样本 $z$ 检验;方差未知但相等用合并方差 $S_w^2=\dfrac{(n_1-1)S_1^2+(n_2-1)S_2^2}{n_1+n_2-2}$ 构造 $t(n_1+n_2-2)$;成对数据则对差值做单样本 $t$ 检验。

**检验步骤.** 立 $H_0,H_1$ → 选统计量 → 定拒绝域(由 $\alpha$ 查分位点) → 代数据计算 → 落入拒绝域则拒绝,给结论。

### 方差的检验(χ²/F 检验)

**(1) 单总体方差:$\chi^2$ 检验.** 设 $X\sim N(\mu,\sigma^2)$,检验 $H_0:\sigma^2=\sigma_0^2$。统计量
$$\chi^2=\frac{(n-1)S^2}{\sigma_0^2}\overset{H_0}{\sim}\chi^2(n-1).$$
双侧拒绝域:$\chi^2>\chi^2_{\alpha/2}(n-1)$ 或 $\chi^2<\chi^2_{1-\alpha/2}(n-1)$。单侧据 $H_1$ 取一端。

**(2) 两总体方差比:$F$ 检验.** 设两独立正态样本,检验 $H_0:\sigma_1^2=\sigma_2^2$。在 $H_0$ 下
$$F=\frac{S_1^2}{S_2^2}\overset{H_0}{\sim}F(n_1-1,\,n_2-1).$$
双侧拒绝域:$F>F_{\alpha/2}(n_1-1,n_2-1)$ 或 $F<F_{1-\alpha/2}(n_1-1,n_2-1)$,其中 $F_{1-\alpha/2}(m,n)=1/F_{\alpha/2}(n,m)$。$F$ 检验常用于"两样本 $t$ 检验前"先验证方差是否相等。

### 拟合优度与独立性检验

前面针对正态总体的参数;**$\chi^2$ 检验**还能检验"分布形状"与"两属性是否独立",属非参数方法。

**拟合优度检验(皮尔逊 $\chi^2$).** 检验样本是否来自某指定分布。把数据分成 $k$ 组,$O_i$ 为第 $i$ 组**观测频数**,$E_i=n p_i$ 为在 $H_0$ 下的**期望频数**($p_i$ 为该组理论概率)。统计量
$$\chi^2=\sum_{i=1}^{k}\frac{(O_i-E_i)^2}{E_i}\overset{H_0}{\sim}\chi^2(k-1-r),$$
其中 $r$ 为由样本估计的参数个数。$\chi^2$ 过大说明观测与理论偏离大,拒绝域 $\chi^2>\chi^2_\alpha(k-1-r)$(右侧)。一般要求各 $E_i\ge5$。

**独立性检验(列联表).** 对 $r\times c$ 列联表,检验两属性是否独立。期望频数 $E_{ij}=\dfrac{\text{(row}_i\text{)}\times\text{(col}_j\text{)}}{n}$(第 $i$ 行和乘以第 $j$ 列和再除以 $n$),统计量
$$\chi^2=\sum_{i,j}\frac{(O_{ij}-E_{ij})^2}{E_{ij}}\overset{H_0}{\sim}\chi^2\big((r-1)(c-1)\big).$$
$\chi^2$ 大则拒绝独立性假设。

### 小结

- $H_0$ vs $H_1$;小概率原理:在 $H_0$ 下罕见事件发生则拒绝 $H_0$。
- 两类错误:弃真 $\alpha=P(\text{reject } H_0\mid H_0)$;取伪 $\beta=P(\text{accept } H_0\mid H_1)$;功效 $1-\beta$。固定 $n$ 时 $\alpha\downarrow\Rightarrow\beta\uparrow$。
- 均值:$\sigma$ 已知 $Z=\frac{\bar X-\mu_0}{\sigma/\sqrt n}\sim N(0,1)$;$\sigma$ 未知 $T=\frac{\bar X-\mu_0}{S/\sqrt n}\sim t(n-1)$。
- 方差:单总体 $\frac{(n-1)S^2}{\sigma_0^2}\sim\chi^2(n-1)$;两总体 $\frac{S_1^2}{S_2^2}\sim F(n_1-1,n_2-1)$。
- 拟合优度/独立性:$\chi^2=\sum\frac{(O-E)^2}{E}$,自由度分别为 $k-1-r$、$(r-1)(c-1)$;右侧拒绝。
- p 值:$p\le\alpha$ 拒绝,$p>\alpha$ 不拒绝;"不拒绝"≠"成立"。

## 典型例题

**例 1.**($\sigma$ 已知,双侧 $z$ 检验)某零件直径 $X\sim N(\mu,\sigma^2)$,标准值 $\mu_0=10$ mm,$\sigma=0.1$ mm 已知。抽 $n=25$ 件得 $\bar X=10.05$ mm。问直径均值是否偏离标准($\alpha=0.05$)?(取 $z_{0.025}=1.96$。)

> **解.** 立 $H_0:\mu=10$,$H_1:\mu\ne10$(双侧)。$\sigma$ 已知,用 $z$ 检验:
> $$Z=\frac{\bar X-\mu_0}{\sigma/\sqrt n}=\frac{10.05-10}{0.1/\sqrt{25}}=\frac{0.05}{0.02}=2.5.$$
> 拒绝域 $|Z|>z_{0.025}=1.96$。因 $|2.5|>1.96$,**落入拒绝域**,故拒绝 $H_0$:在 $0.05$ 水平下认为直径均值显著偏离标准值。

**例 2.**($\sigma$ 未知,单侧 $t$ 检验)某种灯泡寿命 $X\sim N(\mu,\sigma^2)$,$\sigma$ 未知。厂家称平均寿命 $\ge1000$ 小时。抽 $n=16$ 只得 $\bar X=960$,$S=80$。问能否认为平均寿命低于 $1000$ 小时($\alpha=0.05$)?(取 $t_{0.05}(15)=1.753$。)

> **解.** 欲检验"低于",立 $H_0:\mu\ge1000$,$H_1:\mu<1000$(左侧)。$\sigma$ 未知,用 $t$ 检验:
> $$T=\frac{\bar X-\mu_0}{S/\sqrt n}=\frac{960-1000}{80/\sqrt{16}}=\frac{-40}{20}=-2.0.$$
> 左侧拒绝域 $T<-t_{0.05}(15)=-1.753$。因 $-2.0<-1.753$,落入拒绝域,拒绝 $H_0$:在 $0.05$ 水平下认为平均寿命显著低于 $1000$ 小时。

**例 3.**(单总体方差 $\chi^2$ 检验)某仪器测量值 $X\sim N(\mu,\sigma^2)$。要求标准差 $\sigma\le0.5$(即 $\sigma^2\le0.25$)。抽 $n=10$ 得 $S^2=0.45$。问方差是否超标($\alpha=0.05$)?(取 $\chi^2_{0.05}(9)=16.92$。)

> **解.** 立 $H_0:\sigma^2\le0.25$,$H_1:\sigma^2>0.25$(右侧)。统计量
> $$\chi^2=\frac{(n-1)S^2}{\sigma_0^2}=\frac{9\times0.45}{0.25}=\frac{4.05}{0.25}=16.2.$$
> 右侧拒绝域 $\chi^2>\chi^2_{0.05}(9)=16.92$。因 $16.2<16.92$,**未落入拒绝域**,故不拒绝 $H_0$:在 $0.05$ 水平下尚不能认为方差超标。

**例 4.**(两总体方差 $F$ 检验)两台机器加工同一零件,直径都服从正态。分别抽 $n_1=9$、$n_2=7$,得 $S_1^2=0.36$、$S_2^2=0.10$。问两机器加工精度(方差)是否有显著差异($\alpha=0.10$)?(取 $F_{0.05}(8,6)=4.15$,$F_{0.95}(8,6)=1/F_{0.05}(6,8)=1/3.58\approx0.279$。)

> **解.** 立 $H_0:\sigma_1^2=\sigma_2^2$,$H_1:\sigma_1^2\ne\sigma_2^2$(双侧,$\alpha=0.10$,每端 $0.05$)。统计量
> $$F=\frac{S_1^2}{S_2^2}=\frac{0.36}{0.10}=3.6\overset{H_0}{\sim}F(8,6).$$
> 拒绝域 $F>F_{0.05}(8,6)=4.15$ 或 $F<F_{0.95}(8,6)\approx0.279$。因 $0.279<3.6<4.15$,未落入拒绝域,故不拒绝 $H_0$:在 $0.10$ 水平下两机器精度无显著差异。

**例 5.**(拟合优度 $\chi^2$ 检验)掷一颗骰子 $120$ 次,各点出现次数为:$1$ 点 $18$,$2$ 点 $23$,$3$ 点 $16$,$4$ 点 $21$,$5$ 点 $18$,$6$ 点 $24$。问该骰子是否均匀($\alpha=0.05$)?(取 $\chi^2_{0.05}(5)=11.07$。)

> **解.** 立 $H_0$:骰子均匀,各点概率 $p_i=1/6$;$H_1$:不均匀。期望频数 $E_i=np_i=120\times\frac16=20$(每点)。计算
> $$\chi^2=\sum_{i=1}^{6}\frac{(O_i-E_i)^2}{E_i}=\frac{(18\!-\!20)^2+(23\!-\!20)^2+(16\!-\!20)^2+(21\!-\!20)^2+(18\!-\!20)^2+(24\!-\!20)^2}{20}.$$
> 分子 $=4+9+16+1+4+16=50$,故 $\chi^2=50/20=2.5$。自由度 $k-1=6-1=5$(无估计参数,$r=0$)。
> 右侧拒绝域 $\chi^2>\chi^2_{0.05}(5)=11.07$。因 $2.5<11.07$,不拒绝 $H_0$:在 $0.05$ 水平下可认为骰子是均匀的。

## 练习与自测

1. 用文字解释第一类错误与第二类错误的含义,写出 $\alpha,\beta$ 的概率定义,并说明为何固定样本量时不能同时减小二者。
2. 某产品净重 $X\sim N(\mu,\sigma^2)$,标称 $\mu_0=500$ g,$\sigma=8$ g 已知。抽 $n=16$ 得 $\bar X=496$ g。在 $\alpha=0.05$ 下检验净重是否等于 $500$ g(双侧 $z$ 检验)。(取 $z_{0.025}=1.96$。)
3. 某班一次考试成绩 $X\sim N(\mu,\sigma^2)$,$\sigma$ 未知。$n=9$ 个样本得 $\bar X=78$,$S=12$。在 $\alpha=0.05$ 下检验 $H_0:\mu=72$ 对 $H_1:\mu\ne72$($t$ 检验)。(取 $t_{0.025}(8)=2.306$。)
4. 某工序要求方差 $\sigma^2=4$。抽 $n=16$ 得 $S^2=7.5$。在 $\alpha=0.05$ 下检验 $H_0:\sigma^2=4$ 对 $H_1:\sigma^2\ne4$($\chi^2$ 双侧检验)。(取 $\chi^2_{0.025}(15)=27.49$,$\chi^2_{0.975}(15)=6.26$。)
5. 比较两种工艺的方差。甲:$n_1=11$,$S_1^2=8.0$;乙:$n_2=9$,$S_2^2=2.0$。在 $\alpha=0.10$ 下检验 $H_0:\sigma_1^2=\sigma_2^2$ 对 $H_1:\sigma_1^2\ne\sigma_2^2$($F$ 检验)。(取 $F_{0.05}(10,8)=3.35$,$F_{0.95}(10,8)\approx0.326$。)
6. 某遗传实验四类表现型的观测数为 $315,108,101,32$(共 $556$),理论比为 $9:3:3:1$。在 $\alpha=0.05$ 下作拟合优度检验。(取 $\chi^2_{0.05}(3)=7.815$。)

**参考答案.**
1) 第一类错误=$H_0$ 真而拒绝(弃真),$\alpha=P(\text{reject } H_0\mid H_0)$;第二类错误=$H_0$ 假而不拒绝(取伪),$\beta=P(\text{accept } H_0\mid H_1)$;样本量固定时缩小拒绝域(减 $\alpha$)必使不拒绝域增大,从而 $\beta$ 增大,二者此消彼长。
2) $Z=\frac{496-500}{8/\sqrt{16}}=\frac{-4}{2}=-2.0$;$|Z|=2.0>1.96$,落入拒绝域,拒绝 $H_0$,认为净重显著不等于 $500$ g。
3) $T=\frac{78-72}{12/\sqrt9}=\frac{6}{4}=1.5$;$|T|=1.5<t_{0.025}(8)=2.306$,不落入拒绝域,不拒绝 $H_0$。
4) $\chi^2=\frac{(16-1)\times7.5}{4}=\frac{112.5}{4}=28.125$;拒绝域 $\chi^2>27.49$ 或 $<6.26$;因 $28.125>27.49$,拒绝 $H_0$,方差显著不等于 $4$。
5) $F=\frac{S_1^2}{S_2^2}=\frac{8.0}{2.0}=4.0\sim F(10,8)$;拒绝域 $F>3.35$ 或 $F<0.326$;因 $4.0>3.35$,拒绝 $H_0$,两工艺方差有显著差异。
6) 期望频数 $E_i=556\times\{\frac9{16},\frac3{16},\frac3{16},\frac1{16}\}=\{312.75,104.25,104.25,34.75\}$;$\chi^2=\frac{(315-312.75)^2}{312.75}+\frac{(108-104.25)^2}{104.25}+\frac{(101-104.25)^2}{104.25}+\frac{(32-34.75)^2}{34.75}\approx0.016+0.135+0.101+0.218=0.470$;自由度 $4-1=3$;因 $0.470<7.815$,不拒绝 $H_0$,与 $9:3:3:1$ 相符。
